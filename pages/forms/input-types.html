ইনপুট টাইপ (Input Types)

টেক্সট ইনপুট

টাইপ বর্ণনা উদাহরণ
text সাধারণ টেক্সট
password পাসওয়ার্ড (লুকানো)
email ইমেইল ঠিকানা
number সংখ্যা
tel টেলিফোন নম্বর
url ওয়েবসাইট URL

ডেট ও টাইম ইনপুট

টাইপ বর্ণনা উদাহরণ
date তারিখ
time সময়
datetime-local তারিখ ও সময়
month মাস ও বছর
week সপ্তাহ ও বছর

সিলেকশন ইনপুট

টাইপ বর্ণনা উদাহরণ
checkbox চেকবক্স (একাধিক নির্বাচন)
radio রেডিও বাটন (একটি নির্বাচন)
color রং নির্বাচন
range স্লাইডার
file ফাইল আপলোড

বাটন ইনপুট

টাইপ বর্ণনা উদাহরণ
submit ফর্ম জমা দেওয়ার বাটন
reset ফর্ম রিসেট করার বাটন
button সাধারণ বাটন
hidden লুকানো ইনপুট

সম্পূর্ণ উদাহরণ

<form>
  <label>নাম: <input type="text" placeholder="আপনার নাম"></label><br>
  <label>ইমেইল: <input type="email" placeholder="example@email.com"></label><br>
  <label>পাসওয়ার্ড: <input type="password"></label><br>
  <label>জন্ম তারিখ: <input type="date"></label><br>
  <label><input type="checkbox"> শর্তাবলী accept করছি</label><br>
  <input type="submit" value="নিবন্ধন">
</form>
