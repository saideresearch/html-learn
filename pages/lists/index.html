লিস্ট (HTML Lists)

পড়তে সময়: ২৫ মিনিট লেভেল: শিক্ষানবিশ

HTML-এ লিস্ট কী?

HTML-এ লিস্ট হলো একটি গুরুত্বপূর্ণ উপাদান যা তথ্যকে সুসংহত ও সাজানোভাবে উপস্থাপন করতে সাহায্য করে। ওয়েবপৃষ্ঠায় মেনু তৈরি, কন্টেন্টের সূচী, বৈশিষ্ট্য তালিকা, বা যেকোনো ধরণের আইটেমের গ্রুপিংয়ের জন্য লিস্ট ব্যবহার করা হয়। HTML তিনটি প্রধান ধরণের লিস্ট সাপোর্ট করে: আনঅর্ডারড লিস্ট (বুলেটেড), অর্ডারড লিস্ট (নম্বরযুক্ত), এবং ডেসক্রিপশন লিস্ট (টার্ম ও সংজ্ঞা)।

শিখবেন যা:

  • আনঅর্ডারড লিস্ট (ul) ব্যবহার করে বুলেট পয়েন্ট তৈরি
  • অর্ডারড লিস্ট (ol) ব্যবহার করে সংখ্যা বা রোমান সংখ্যাসহ লিস্ট
  • ডেসক্রিপশন লিস্ট (dl) ব্যবহার করে শব্দ ও তার সংজ্ঞা দেখানো
  • নেস্টেড লিস্ট তৈরি করে জটিল তথ্য উপস্থাপন
  • CSS দিয়ে লিস্টের ডিজাইন ও স্টাইলিং

আনঅর্ডারড লিস্ট (Unordered List - ul)

<ul> ট্যাগ ব্যবহার করে আনঅর্ডারড লিস্ট তৈরি করা হয়। এতে প্রতিটি আইটেমের আগে বুলেট পয়েন্ট (●, ○, ■) দেখায়। এই লিস্ট সাধারণত এমন জায়গায় ব্যবহার করা হয় যেখানে আইটেমগুলোর ক্রম গুরুত্বপূর্ণ নয়।

আনঅর্ডারড লিস্ট উদাহরণ:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
  • HTML
  • CSS
  • JavaScript

বিভিন্ন বুলেট স্টাইল (CSS সহ):

<ul style="list-style-type: square;">
  <li>স্কয়ার বুলেট</li>
</ul>
<ul style="list-style-type: circle;">
  <li>সার্কেল বুলেট</li>
</ul>
  • স্কয়ার বুলেট
  • সার্কেল বুলেট

অর্ডারড লিস্ট (Ordered List - ol)

<ol> ট্যাগ ব্যবহার করে অর্ডারড লিস্ট তৈরি করা হয়। এতে প্রতিটি আইটেমের আগে সংখ্যা (1, 2, 3), রোমান সংখ্যা (I, II, III), বা অক্ষর (A, B, C) দেখায়। এই লিস্ট ব্যবহার করা হয় যেখানে ক্রম বা র্যাংকিং গুরুত্বপূর্ণ।

অর্ডারড লিস্ট উদাহরণ:

<ol>
  <li>>প্রথম ধাপ</li>
  <li>দ্বিতীয় ধাপ</li>
  <li>তৃতীয় ধাপ</li>
</ol>
  1. প্রথম ধাপ
  2. দ্বিতীয় ধাপ
  3. তৃতীয় ধাপ

type অ্যাট্রিবিউট দিয়ে বিভিন্ন স্টাইল:

<ol type="A">
  <li>বড় হাতের অক্ষর</li>
</ol>
<ol type="i">
  <li>ছোট হাতের রোমান</li>
</ol>
<ol start="5">
  <li>৫ থেকে শুরু</li>
</ol>
  1. বড় হাতের অক্ষর
  1. ছোট হাতের রোমান
  1. ৫ থেকে শুরু

ডেসক্রিপশন লিস্ট (Description List - dl)

<dl> (description list), <dt> (description term), এবং <dd> (description definition) ট্যাগ ব্যবহার করে ডেসক্রিপশন লিস্ট তৈরি করা হয়। এটি শব্দ বা টার্ম এবং তার সংজ্ঞা বা বিবরণ দেখানোর জন্য আদর্শ।

ডেসক্রিপশন লিস্ট উদাহরণ:

<dl>
  <dt>HTML</dt>
  <dd>হাইপারটেক্সট মার্কআপ ল্যাঙ্গুয়েজ - ওয়েবপৃষ্ঠার কাঠামো তৈরি করে।</dd>
  <dt>CSS</dt>
  <dd>ক্যাসকেডিং স্টাইল শিট - ওয়েবপৃষ্ঠার ডিজাইন নির্ধারণ করে।</dd>
  <dt>JavaScript</dt>
  <dd>ওয়েবপৃষ্ঠায় ইন্টারঅ্যাকটিভিটি যোগ করে।</dd>
</dl>
HTML
হাইপারটেক্সট মার্কআপ ল্যাঙ্গুয়েজ - ওয়েবপৃষ্ঠার কাঠামো তৈরি করে।
CSS
ক্যাসকেডিং স্টাইল শিট - ওয়েবপৃষ্ঠার ডিজাইন নির্ধারণ করে।
JavaScript
ওয়েবপৃষ্ঠায় ইন্টারঅ্যাকটিভিটি যোগ করে।

নেস্টেড লিস্ট (Nested Lists)

একটি লিস্টের ভিতরে আরেকটি লিস্ট তৈরি করাকে নেস্টেড লিস্ট বলে। এটি ব্যবহার করে সাব-ক্যাটাগরি, মেনু বা জটিল তথ্য উপস্থাপন করা যায়। নেস্টেড লিস্ট আনঅর্ডারড, অর্ডারড বা ডেসক্রিপশন লিস্টের ভিতরে যেকোনো ধরনের লিস্ট হতে পারে।

নেস্টেড লিস্ট উদাহরণ:

<ul>
  <li>ফ্রন্ট-এন্ড ডেভেলপমেন্ট
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>ব্যাক-এন্ড ডেভেলপমেন্ট
    <ul>
      <li>PHP</li>
      <li>Python</li>
      <li>Node.js</li>
    </ul>
  </li>
</ul>
  • ফ্রন্ট-এন্ড ডেভেলপমেন্ট
    • HTML
    • CSS
    • JavaScript
  • ব্যাক-এন্ড ডেভেলপমেন্ট
    • PHP
    • Python
    • Node.js

লিস্ট স্টাইলিং (CSS দিয়ে)

CSS ব্যবহার করে লিস্টের চেহারা সম্পূর্ণ কাস্টমাইজ করা যায়। বুলেট বা নম্বরের ধরন পরিবর্তন, ইমেজ বুলেট ব্যবহার, মার্জিন, প্যাডিং, এবং হোভার ইফেক্ট যোগ করা সম্ভব।

কাস্টম বুলেট ইমেজ:

<ul style="list-style-image: url('arrow.png');">
  <li>কাস্টম বুলেট সহ আইটেম</li>
  <li>আরেকটি আইটেম</li>
</ul>

<ul style="list-style: none; padding-left: 0;">
  <li>✓ বুলেট ছাড়া লিস্ট</li>
  <li>✓ কাস্টম সিম্বল ব্যবহার</li>
</ul>
  • ✓ বুলেট ছাড়া লিস্ট
  • ✓ কাস্টম সিম্বল ব্যবহার

লিস্ট স্টাইলিং CSS প্রপার্টি:

  • list-style-type – বুলেট বা নম্বরের ধরন (disc, circle, square, decimal, lower-roman, ইত্যাদি)
  • list-style-image – কাস্টম ইমেজ বুলেট ব্যবহার
  • list-style-position – বুলেটের অবস্থান (inside/outside)
  • list-style – শর্টহ্যান্ড প্রপার্টি

লিস্টের ব্যবহারিক উদাহরণ

ওয়েব ডেভেলপমেন্টে লিস্ট বিভিন্নভাবে ব্যবহার করা হয়। নিচে কিছু সাধারণ ব্যবহার দেখানো হলো:

নেভিগেশন মেনু (CSS সহ):

<ul class="nav-menu">
  <li><a href="#">হোম</a></li>
  <li><a href="#">ব্লগ</a></li>
  <li><a href="#">যোগাযোগ</a></li>
</ul>
  • হোম
  • ব্লগ
  • যোগাযোগ

টু-ডু লিস্ট (টাস্ক লিস্ট):

<ul style="list-style: none;">
  <li>☐ HTML শিখা</li>
  <li>☑ CSS শিখা (সম্পন্ন)</li>
  <li>☐ JavaScript শিখা</li>
</ul>
  • ☐ HTML শিখা
  • ☑ CSS শিখা (সম্পন্ন)
  • ☐ JavaScript শিখা

সম্পূর্ণ টপিক লিস্ট

নিচের লিংকগুলোতে ক্লিক করে প্রতিটি টপিক বিস্তারিত শিখুন:

📋 আনঅর্ডারড লিস্ট (ul)

বুলেট পয়েন্ট ব্যবহার করে লিস্ট তৈরি করা শিখুন

🔢 অর্ডারড লিস্ট (ol)

সংখ্যা, রোমান সংখ্যা বা অক্ষর দিয়ে সাজানো লিস্ট

📖 ডেসক্রিপশন লিস্ট (dl)

শব্দ ও তার সংজ্ঞা উপস্থাপনের আদর্শ পদ্ধতি

🌳 নেস্টেড লিস্ট

লিস্টের ভিতরে লিস্ট তৈরি করে জটিল তথ্য উপস্থাপন

🎨 লিস্ট স্টাইলিং
