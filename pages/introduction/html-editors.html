HTML এডিটর (HTML Editors)

পড়তে সময়: ১২ মিনিট লেভেল: শিক্ষানবিশ

HTML এডিটর কী?

HTML এডিটর হলো এমন একটি সফটওয়্যার বা অ্যাপ্লিকেশন যা ব্যবহার করে আপনি HTML কোড লিখতে, সম্পাদনা করতে এবং সংরক্ষণ করতে পারেন। ভালো একটি এডিটর আপনার কোডিং অভিজ্ঞতা অনেক সহজ ও উপভোগ্য করে তোলে।

মূলত দুই ধরনের HTML এডিটর আছে:

  • টেক্সট-ভিত্তিক এডিটর: শুধুমাত্র টেক্সট এডিট করার সুবিধা (যেমন: নোটপ্যাড)
  • WYSIWYG এডিটর: What You See Is What You Get - আপনি যা দেখবেন তাই পাবেন (যেমন: Dreamweaver)

সাধারণ টেক্সট এডিটর

শিক্ষানবিশদের জন্য সাধারণ টেক্সট এডিটর দিয়ে শুরু করা সবচেয়ে ভালো। এতে HTML-এর মৌলিক কাঠামো ও নিয়মগুলো ভালোভাবে বোঝা যায়।

উইন্ডোজের জন্য:

এডিটরের নাম বৈশিষ্ট্য মূল্য
নোটপ্যাড (Notepad) অপারেটিং সিস্টেমের সাথে বিনামূল্যে পাওয়া যায়, খুবই সহজ ফ্রি
নোটপ্যাড++ (Notepad++) সিনট্যাক্স হাইলাইটিং, একাধিক ট্যাব, প্লাগইন সাপোর্ট ফ্রি
সাবলাইম টেক্সট (Sublime Text) দ্রুত, শক্তিশালী, কাস্টমাইজযোগ্য $৯৯ (আনলিমিটেড ট্রায়াল)

ম্যাকের জন্য:

এডিটরের নাম বৈশিষ্ট্য মূল্য
টেক্সটএডিট (TextEdit) অপারেটিং সিস্টেমের সাথে বিনামূল্যে পাওয়া যায় ফ্রি
ব্র্যাকেটস (Brackets) অ্যাডোবি তৈরি করেছে, লাইভ প্রিভিউ সুবিধা ফ্রি
কোড (Code) - ভিজুয়াল স্টুডিও মাইক্রোসফটের বিনামূল্যের এডিটর ফ্রি

লিনাক্সের জন্য:

এডিটরের নাম বৈশিষ্ট্য মূল্য
জিনিট (Gedit) ডেস্কটপ পরিবেশের সাথে আসে, সহজ ও হালকা ফ্রি
ভিম (Vim) টার্মিনাল-ভিত্তিক, কীবোর্ড শর্টকাটে চলে ফ্রি
এটম (Atom) গিটহাব তৈরি করেছে, প্যাকেজ সমৃদ্ধ ফ্রি

জনপ্রিয় পেশাদার এডিটর

পেশাদার ওয়েব ডেভেলপাররা সাধারণত শক্তিশালী ফিচারসমৃদ্ধ এডিটর ব্যবহার করেন। নিচে কয়েকটি জনপ্রিয় পেশাদার এডিটরের বিবরণ দেওয়া হলো।

ভিজুয়াল স্টুডিও কোড (VS Code)

বর্তমানে সবচেয়ে জনপ্রিয় ও শক্তিশালী ফ্রি এডিটর। মাইক্রোসফট তৈরি করেছে।

বৈশিষ্ট্য:

  • ইন্টেলিসেন্স - কোড অটো-কমপ্লিশন
  • ইন্টিগ্রেটেড টার্মিনাল
  • গিট কন্ট্রোল
  • হাজার হাজার এক্সটেনশন
  • ডিবাগিং সাপোর্ট
  • লাইভ শেয়ার (সহযোগিতামূলক সম্পাদনা)

অ্যাডোব ড্রিমউইভার (Adobe Dreamweaver)

পেশাদার WYSIWYG এডিটর। কোড ও ডিজাইন দুই ভিউতেই কাজ করা যায়।

বৈশিষ্ট্য:

  • স্প্লিট ভিউ - একসাথে কোড ও ডিজাইন দেখা
  • ফটোশপের সাথে ইন্টিগ্রেশন
  • সার্ভার-সাইড অন্তর্ভুক্তি
  • সিএসএস ডিজাইন টুলস

অনলাইন HTML এডিটর

অনলাইন এডিটর ব্যবহারের জন্য কোনো সফটওয়্যার ইনস্টল করার প্রয়োজন নেই। ব্রাউজার খুললেই কোড লেখা শুরু করা যায়। শিক্ষানবিশ ও দ্রুত পরীক্ষা-নিরীক্ষার জন্য এগুলো খুবই উপযোগী।

এডিটরের নাম বৈশিষ্ট্য ওয়েবসাইট
CodePen HTML, CSS, JavaScript একসাথে লেখা ও ফলাফল দেখা, কমিউনিটি শেয়ারিং codepen.io
JSFiddle ফ্রেমওয়ার্ক সাপোর্ট, সহযোগিতামূলক সম্পাদনা jsfiddle.net
JS Bin লাইভ প্রিভিউ, কনসোল লগ দেখা jsbin.com
Replit পূর্ণাঙ্গ উন্নয়ন পরিবেশ, হোস্টিং সুবিধা replit.com
OneCompiler বাংলা ভাষায় টিউটোরিয়াল সহ, সহজ ইন্টারফেস onecompiler.com

মোবাইল HTML এডিটর

বর্তমানে স্মার্টফোন দিয়েও HTML কোড লেখা সম্ভব। নিচে কয়েকটি জনপ্রিয় মোবাইল অ্যাপের তালিকা দেওয়া হলো।

অ্যান্ড্রয়েডের জন্য:

  • Spck Editor: গিট ইন্টিগ্রেশন, সিনট্যাক্স হাইলাইটিং
  • Dcoder: ৫০+ ভাষা সাপোর্ট, কম্পাইলার
  • QuickEdit: হালকা, দ্রুত, রুট সাপোর্ট

আইওএসের জন্য:

  • Koder: ওয়েব ডেভেলপমেন্ট, FTP/SFTP সাপোর্ট
  • Textastic: শক্তিশালী কোড এডিটর, সিনট্যাক্স হাইলাইটিং
  • Pythonista: পাইথনের পাশাপাশি HTML সাপোর্ট

বাংলাদেশে জনপ্রিয় HTML এডিটর

বাংলাদেশের ওয়েব ডেভেলপারদের মধ্যে কিছু এডিটর বিশেষভাবে জনপ্রিয়। নিচে তাদের তালিকা ও ব্যবহারের কারণ দেওয়া হলো।

এডিটর জনপ্রিয়তার কারণ ব্যবহারকারী স্তর
VS Code ফ্রি, বাংলা টিউটোরিয়াল প্রচুর, এক্সটেনশন সমৃদ্ধ শিক্ষানবিশ-পেশাদার
Sublime Text দ্রুত, হালকা, পুরনো কম্পিউটারে চলে মাধ্যমিক-পেশাদার
Notepad++ অনেক পুরনো, স্কুল-কলেজে শেখানো হয় শিক্ষানবিশ
CodePen অনলাইন, ফ্রিল্যান্সাররা পোর্টফোলিও দেখায় সকল স্তর

কীভাবে সঠিক এডিটর নির্বাচন করবেন

আপনার জন্য কোন এডিটর সঠিক, তা নির্ভর করে কয়েকটি বিষয়ের উপর। নিচের প্রশ্নগুলো নিজেকে করুন:

  1. আপনার কম্পিউটারের স্পেসিফিকেশন কেমন? - পুরনো কম্পিউটার হলে হালকা এডিটর নিন
  2. আপনি কি শিক্ষানবিশ? - তবে সাধারণ এডিটর দিয়ে শুরু করুন
  3. কী ধরনের প্রকল্প করবেন? - বড় প্রকল্পের জন্য শক্তিশালী এডিটর প্রয়োজন
  4. বাজেট কত? - ফ্রি এডিটর দিয়েও পেশাদার কাজ করা যায়
  5. অফলাইন না অনলাইনে কাজ করবেন? - অনলাইন এডিটর ইনস্টলেশনের ঝামেলা মুক্ত

শিক্ষানবিশদের জন্য পরামর্শ:

প্রথমে VS Code দিয়ে শুরু করুন। কারণ:

  • বিনামূল্যে
  • বাংলায় অসংখ্য টিউটোরিয়াল আছে
  • ইনস্টল করা সহজ
  • এক্সটেনশনের মাধ্যমে ধীরে ধীরে শক্তি বাড়ানো যায়
  • ইন্ডাস্ট্রি স্ট্যান্ডার্ড

VS Code ইনস্টল ও সেটআপ গাইড

ধাপ ১: ডাউনলোড

code.visualstudio.com থেকে আপনার অপারেটিং সিস্টেমের জন্য VS Code ডাউনলোড করুন।

ধাপ ২: ইনস্টল

ডাউনলোড করা ফাইলটি খুলে Next, Next দিয়ে ইনস্টল করুন। ইনস্টলের সময় "Add to PATH" অপশনটি চেক রাখুন।

ধাপ ৩: প্রয়োজনীয় এক্সটেনশন

VS Code খুলে বাম পাশের এক্সটেনশন আইকন (বা Ctrl+Shift+X) চাপুন। নিচের এক্সটেনশনগুলো ইনস্টল করুন:

এক্সটেনশনের নাম কাজ
Live Server HTML ফাইল ব্রাউজারে লাইভ দেখায় (সেভ দিলেই আপডেট)
Prettier কোড ফরম্যাট করে সুন্দর করে
HTML CSS Support HTML-এ CSS ক্লাস অটো-সাজেস্ট করে
Auto Rename Tag শুরু ট্যাগ পরিবর্তন করলে শেষ ট্যাগ অটো পরিবর্তন হয়

নোটপ্যাড দিয়ে প্রথম HTML পৃষ্ঠা

শিক্ষানবিশদের জন্য নোটপ্যাড দিয়ে শুরু করার পদ্ধতি:

  1. নোটপ্যাড খুলুন: Start মেনুতে গিয়ে Notepad লিখে সার্চ করুন
  2. HTML কোড লিখুন: 
    <!DOCTYPE html>
<html>
<head>
  <title>আমার প্রথম পৃষ্ঠা</title>
</head>
<body>
  <h1>স্বাগতম!</h1>
  <p>এটি আমার প্রথম HTML পৃষ্ঠা।</p>
</body>
</html>
  3. ফাইল সংরক্ষণ করুন: File → Save As → ফাইল টাইপ "All Files (*.*)" নির্বাচন করুন → ফাইলের নাম "index.html" দিন → Save
  4. ব্রাউজারে খুলুন: যেখানে সেভ করেছেন, সেই index.html ফাইলটি ডাবল ক্লিক করে খুলুন

এডিটরের প্রয়োজনীয় বৈশিষ্ট্য

একটি ভালো HTML এডিটরে নিচের বৈশিষ্ট্যগুলো থাকা উচিত:

বৈশিষ্ট্য কেন প্রয়োজন
সিনট্যাক্স হাইলাইটিং কোডের বিভিন্ন অংশ রঙিন দেখায়, পড়তে ও বুঝতে সুবিধা হয়
অটো-কমপ্লিশন ট্যাগ লেখার সময় সাজেশন দেখায়, টাইপিং দ্রুত হয় ও ভুল কমে
লাইন নাম্বারিং প্রতিটি লাইনের পাশে নাম্বার দেখায়, ডিবাগিং সহজ করে
মাল্টিপল কার্সর একসাথে অনেক জায়গায় এডিট করা যায়
ফাইন্ড এন্ড রিপ্লেস কোডের মধ্যে খোঁজা ও পরিবর্তন করা সহজ করে
প্লাগইন/এক্সটেনশন প্রয়োজন অনুযায়ী নতুন ফিচার যুক্ত করা যায়

অনুশীলনী

  1. টেক্সট-ভিত্তিক এডিটর ও WYSIWYG এডিটরের মধ্যে তিনটি পার্থক্য লেখ।
  2. VS Code-এর পাঁচটি বৈশিষ্ট্যের নাম লেখ।
  3. তিনটি জনপ্রিয় অনলাইন HTML এডিটরের নাম ও তাদের বৈশিষ্ট্য লেখ।
  4. শিক্ষানবিশদের জন্য কোন এডিটর সবচেয়ে ভালো এবং কেন?
  5. নোটপ্যাড দিয়ে একটি HTML ফাইল কীভাবে তৈরি করবেন, ধাপগুলো লেখ।
  6. নিচের এডিটরগুলোর মধ্যে কোনটি WYSIWYG? (Dreamweaver, VS Code, Notepad++, CodePen)
সমাধান দেখুন
  1. টেক্সট-ভিত্তিক: কোড দেখায়, হালকা, বেশি নিয়ন্ত্রণ; WYSIWYG: ডিজাইন দেখায়, সহজ, স্বয়ংক্রিয় কোড
  2. ইন্টেলিসেন্স, ইন্টিগ্রেটেড টার্মিনাল, গিট কন্ট্রোল, এক্সটেনশন, ডিবাগিং
  3. CodePen (লাইভ প্রিভিউ, কমিউনিটি), JSFiddle (ফ্রেমওয়ার্ক সাপোর্ট), Replit (পূর্ণাঙ্গ পরিবেশ, হোস্টিং)
  4. VS Code - কারণ ফ্রি, বাংলা টিউটোরিয়াল আছে, ইন্ডাস্ট্রি স্ট্যান্ডার্ড, এক্সটেনশন সমৃদ্ধ
  5. নোটপ্যাড খুলুন → কোড লিখুন → File → Save As → ফাইল টাইপ All Files → index.html নামে সেভ → ব্রাউজারে খুলুন
  6. Dreamweaver (এটি WYSIWYG)

সতর্কতা ও টিপস

⚠️ সতর্কতা: ওয়ার্ড প্রসেসর (যেমন: Microsoft Word) দিয়ে কখনো HTML কোড লিখবেন না। এতে অদৃশ্য ফরম্যাটিং কোড যোগ হয়ে যায় যা আপনার HTML নষ্ট করতে পারে।

💡 টিপস: VS Code-এ "! + Tab" চাপলেই HTML5-এর মৌলিক কাঠামো অটোমেটিক তৈরি হয়ে যায়। এটি আপনার সময় বাঁচাবে।

💡 টিপস: কোড লেখার সময় নিয়মিত "Ctrl+S" চেপে ফাইল সেভ করার অভ্যাস করুন। অপ্রত্যাশিত বন্ধ হয়ে গেলে আপনার কাজ নষ্ট হবে না।

HTML এডিটর (HTML Editors)

পড়তে সময়: ১২ মিনিট লেভেল: শিক্ষানবিশ

HTML এডিটর কী?

HTML এডিটর হলো এমন একটি সফটওয়্যার বা অ্যাপ্লিকেশন যা ব্যবহার করে আপনি HTML কোড লিখতে, সম্পাদনা করতে এবং সংরক্ষণ করতে পারেন। ভালো একটি এডিটর আপনার কোডিং অভিজ্ঞতা অনেক সহজ ও উপভোগ্য করে তোলে।

মূলত দুই ধরনের HTML এডিটর আছে:

  • টেক্সট-ভিত্তিক এডিটর: শুধুমাত্র টেক্সট এডিট করার সুবিধা (যেমন: নোটপ্যাড)
  • WYSIWYG এডিটর: What You See Is What You Get - আপনি যা দেখবেন তাই পাবেন (যেমন: Dreamweaver)

সাধারণ টেক্সট এডিটর

শিক্ষানবিশদের জন্য সাধারণ টেক্সট এডিটর দিয়ে শুরু করা সবচেয়ে ভালো। এতে HTML-এর মৌলিক কাঠামো ও নিয়মগুলো ভালোভাবে বোঝা যায়।

উইন্ডোজের জন্য:

এডিটরের নাম বৈশিষ্ট্য মূল্য
নোটপ্যাড (Notepad) অপারেটিং সিস্টেমের সাথে বিনামূল্যে পাওয়া যায়, খুবই সহজ ফ্রি
নোটপ্যাড++ (Notepad++) সিনট্যাক্স হাইলাইটিং, একাধিক ট্যাব, প্লাগইন সাপোর্ট ফ্রি
সাবলাইম টেক্সট (Sublime Text) দ্রুত, শক্তিশালী, কাস্টমাইজযোগ্য $৯৯ (আনলিমিটেড ট্রায়াল)

ম্যাকের জন্য:

এডিটরের নাম বৈশিষ্ট্য মূল্য
টেক্সটএডিট (TextEdit) অপারেটিং সিস্টেমের সাথে বিনামূল্যে পাওয়া যায় ফ্রি
ব্র্যাকেটস (Brackets) অ্যাডোবি তৈরি করেছে, লাইভ প্রিভিউ সুবিধা ফ্রি
কোড (Code) - ভিজুয়াল স্টুডিও মাইক্রোসফটের বিনামূল্যের এডিটর ফ্রি

লিনাক্সের জন্য:

এডিটরের নাম বৈশিষ্ট্য মূল্য
জিনিট (Gedit) ডেস্কটপ পরিবেশের সাথে আসে, সহজ ও হালকা ফ্রি
ভিম (Vim) টার্মিনাল-ভিত্তিক, কীবোর্ড শর্টকাটে চলে ফ্রি
এটম (Atom) গিটহাব তৈরি করেছে, প্যাকেজ সমৃদ্ধ ফ্রি

জনপ্রিয় পেশাদার এডিটর

পেশাদার ওয়েব ডেভেলপাররা সাধারণত শক্তিশালী ফিচারসমৃদ্ধ এডিটর ব্যবহার করেন। নিচে কয়েকটি জনপ্রিয় পেশাদার এডিটরের বিবরণ দেওয়া হলো।

ভিজুয়াল স্টুডিও কোড (VS Code)

বর্তমানে সবচেয়ে জনপ্রিয় ও শক্তিশালী ফ্রি এডিটর। মাইক্রোসফট তৈরি করেছে।

বৈশিষ্ট্য:

  • ইন্টেলিসেন্স - কোড অটো-কমপ্লিশন
  • ইন্টিগ্রেটেড টার্মিনাল
  • গিট কন্ট্রোল
  • হাজার হাজার এক্সটেনশন
  • ডিবাগিং সাপোর্ট
  • লাইভ শেয়ার (সহযোগিতামূলক সম্পাদনা)

অ্যাডোব ড্রিমউইভার (Adobe Dreamweaver)

পেশাদার WYSIWYG এডিটর। কোড ও ডিজাইন দুই ভিউতেই কাজ করা যায়।

বৈশিষ্ট্য:

  • স্প্লিট ভিউ - একসাথে কোড ও ডিজাইন দেখা
  • ফটোশপের সাথে ইন্টিগ্রেশন
  • সার্ভার-সাইড অন্তর্ভুক্তি
  • সিএসএস ডিজাইন টুলস

অনলাইন HTML এডিটর

অনলাইন এডিটর ব্যবহারের জন্য কোনো সফটওয়্যার ইনস্টল করার প্রয়োজন নেই। ব্রাউজার খুললেই কোড লেখা শুরু করা যায়। শিক্ষানবিশ ও দ্রুত পরীক্ষা-নিরীক্ষার জন্য এগুলো খুবই উপযোগী।

এডিটরের নাম বৈশিষ্ট্য ওয়েবসাইট
CodePen HTML, CSS, JavaScript একসাথে লেখা ও ফলাফল দেখা, কমিউনিটি শেয়ারিং codepen.io
JSFiddle ফ্রেমওয়ার্ক সাপোর্ট, সহযোগিতামূলক সম্পাদনা jsfiddle.net
JS Bin লাইভ প্রিভিউ, কনসোল লগ দেখা jsbin.com
Replit পূর্ণাঙ্গ উন্নয়ন পরিবেশ, হোস্টিং সুবিধা replit.com
OneCompiler বাংলা ভাষায় টিউটোরিয়াল সহ, সহজ ইন্টারফেস onecompiler.com

মোবাইল HTML এডিটর

বর্তমানে স্মার্টফোন দিয়েও HTML কোড লেখা সম্ভব। নিচে কয়েকটি জনপ্রিয় মোবাইল অ্যাপের তালিকা দেওয়া হলো।

অ্যান্ড্রয়েডের জন্য:

  • Spck Editor: গিট ইন্টিগ্রেশন, সিনট্যাক্স হাইলাইটিং
  • Dcoder: ৫০+ ভাষা সাপোর্ট, কম্পাইলার
  • QuickEdit: হালকা, দ্রুত, রুট সাপোর্ট

আইওএসের জন্য:

  • Koder: ওয়েব ডেভেলপমেন্ট, FTP/SFTP সাপোর্ট
  • Textastic: শক্তিশালী কোড এডিটর, সিনট্যাক্স হাইলাইটিং
  • Pythonista: পাইথনের পাশাপাশি HTML সাপোর্ট

বাংলাদেশে জনপ্রিয় HTML এডিটর

বাংলাদেশের ওয়েব ডেভেলপারদের মধ্যে কিছু এডিটর বিশেষভাবে জনপ্রিয়। নিচে তাদের তালিকা ও ব্যবহারের কারণ দেওয়া হলো।

এডিটর জনপ্রিয়তার কারণ ব্যবহারকারী স্তর
VS Code ফ্রি, বাংলা টিউটোরিয়াল প্রচুর, এক্সটেনশন সমৃদ্ধ শিক্ষানবিশ-পেশাদার
Sublime Text দ্রুত, হালকা, পুরনো কম্পিউটারে চলে মাধ্যমিক-পেশাদার
Notepad++ অনেক পুরনো, স্কুল-কলেজে শেখানো হয় শিক্ষানবিশ
CodePen অনলাইন, ফ্রিল্যান্সাররা পোর্টফোলিও দেখায় সকল স্তর

কীভাবে সঠিক এডিটর নির্বাচন করবেন

আপনার জন্য কোন এডিটর সঠিক, তা নির্ভর করে কয়েকটি বিষয়ের উপর। নিচের প্রশ্নগুলো নিজেকে করুন:

  1. আপনার কম্পিউটারের স্পেসিফিকেশন কেমন? - পুরনো কম্পিউটার হলে হালকা এডিটর নিন
  2. আপনি কি শিক্ষানবিশ? - তবে সাধারণ এডিটর দিয়ে শুরু করুন
  3. কী ধরনের প্রকল্প করবেন? - বড় প্রকল্পের জন্য শক্তিশালী এডিটর প্রয়োজন
  4. বাজেট কত? - ফ্রি এডিটর দিয়েও পেশাদার কাজ করা যায়
  5. অফলাইন না অনলাইনে কাজ করবেন? - অনলাইন এডিটর ইনস্টলেশনের ঝামেলা মুক্ত

শিক্ষানবিশদের জন্য পরামর্শ:

প্রথমে VS Code দিয়ে শুরু করুন। কারণ:

  • বিনামূল্যে
  • বাংলায় অসংখ্য টিউটোরিয়াল আছে
  • ইনস্টল করা সহজ
  • এক্সটেনশনের মাধ্যমে ধীরে ধীরে শক্তি বাড়ানো যায়
  • ইন্ডাস্ট্রি স্ট্যান্ডার্ড

VS Code ইনস্টল ও সেটআপ গাইড

ধাপ ১: ডাউনলোড

code.visualstudio.com থেকে আপনার অপারেটিং সিস্টেমের জন্য VS Code ডাউনলোড করুন।

ধাপ ২: ইনস্টল

ডাউনলোড করা ফাইলটি খুলে Next, Next দিয়ে ইনস্টল করুন। ইনস্টলের সময় "Add to PATH" অপশনটি চেক রাখুন।

ধাপ ৩: প্রয়োজনীয় এক্সটেনশন

VS Code খুলে বাম পাশের এক্সটেনশন আইকন (বা Ctrl+Shift+X) চাপুন। নিচের এক্সটেনশনগুলো ইনস্টল করুন:

এক্সটেনশনের নাম কাজ
Live Server HTML ফাইল ব্রাউজারে লাইভ দেখায় (সেভ দিলেই আপডেট)
Prettier কোড ফরম্যাট করে সুন্দর করে
HTML CSS Support HTML-এ CSS ক্লাস অটো-সাজেস্ট করে
Auto Rename Tag শুরু ট্যাগ পরিবর্তন করলে শেষ ট্যাগ অটো পরিবর্তন হয়

নোটপ্যাড দিয়ে প্রথম HTML পৃষ্ঠা

শিক্ষানবিশদের জন্য নোটপ্যাড দিয়ে শুরু করার পদ্ধতি:

  1. নোটপ্যাড খুলুন: Start মেনুতে গিয়ে Notepad লিখে সার্চ করুন
  2. HTML কোড লিখুন: 
    <!DOCTYPE html>
<html>
<head>
  <title>আমার প্রথম পৃষ্ঠা</title>
</head>
<body>
  <h1>স্বাগতম!</h1>
  <p>এটি আমার প্রথম HTML পৃষ্ঠা।</p>
</body>
</html>
  3. ফাইল সংরক্ষণ করুন: File → Save As → ফাইল টাইপ "All Files (*.*)" নির্বাচন করুন → ফাইলের নাম "index.html" দিন → Save
  4. ব্রাউজারে খুলুন: যেখানে সেভ করেছেন, সেই index.html ফাইলটি ডাবল ক্লিক করে খুলুন

এডিটরের প্রয়োজনীয় বৈশিষ্ট্য

একটি ভালো HTML এডিটরে নিচের বৈশিষ্ট্যগুলো থাকা উচিত:

বৈশিষ্ট্য কেন প্রয়োজন
সিনট্যাক্স হাইলাইটিং কোডের বিভিন্ন অংশ রঙিন দেখায়, পড়তে ও বুঝতে সুবিধা হয়
অটো-কমপ্লিশন ট্যাগ লেখার সময় সাজেশন দেখায়, টাইপিং দ্রুত হয় ও ভুল কমে
লাইন নাম্বারিং প্রতিটি লাইনের পাশে নাম্বার দেখায়, ডিবাগিং সহজ করে
মাল্টিপল কার্সর একসাথে অনেক জায়গায় এডিট করা যায়
ফাইন্ড এন্ড রিপ্লেস কোডের মধ্যে খোঁজা ও পরিবর্তন করা সহজ করে
প্লাগইন/এক্সটেনশন প্রয়োজন অনুযায়ী নতুন ফিচার যুক্ত করা যায়

অনুশীলনী

  1. টেক্সট-ভিত্তিক এডিটর ও WYSIWYG এডিটরের মধ্যে তিনটি পার্থক্য লেখ।
  2. VS Code-এর পাঁচটি বৈশিষ্ট্যের নাম লেখ।
  3. তিনটি জনপ্রিয় অনলাইন HTML এডিটরের নাম ও তাদের বৈশিষ্ট্য লেখ।
  4. শিক্ষানবিশদের জন্য কোন এডিটর সবচেয়ে ভালো এবং কেন?
  5. নোটপ্যাড দিয়ে একটি HTML ফাইল কীভাবে তৈরি করবেন, ধাপগুলো লেখ।
  6. নিচের এডিটরগুলোর মধ্যে কোনটি WYSIWYG? (Dreamweaver, VS Code, Notepad++, CodePen)
সমাধান দেখুন
  1. টেক্সট-ভিত্তিক: কোড দেখায়, হালকা, বেশি নিয়ন্ত্রণ; WYSIWYG: ডিজাইন দেখায়, সহজ, স্বয়ংক্রিয় কোড
  2. ইন্টেলিসেন্স, ইন্টিগ্রেটেড টার্মিনাল, গিট কন্ট্রোল, এক্সটেনশন, ডিবাগিং
  3. CodePen (লাইভ প্রিভিউ, কমিউনিটি), JSFiddle (ফ্রেমওয়ার্ক সাপোর্ট), Replit (পূর্ণাঙ্গ পরিবেশ, হোস্টিং)
  4. VS Code - কারণ ফ্রি, বাংলা টিউটোরিয়াল আছে, ইন্ডাস্ট্রি স্ট্যান্ডার্ড, এক্সটেনশন সমৃদ্ধ
  5. নোটপ্যাড খুলুন → কোড লিখুন → File → Save As → ফাইল টাইপ All Files → index.html নামে সেভ → ব্রাউজারে খুলুন
  6. Dreamweaver (এটি WYSIWYG)

সতর্কতা ও টিপস

⚠️ সতর্কতা: ওয়ার্ড প্রসেসর (যেমন: Microsoft Word) দিয়ে কখনো HTML কোড লিখবেন না। এতে অদৃশ্য ফরম্যাটিং কোড যোগ হয়ে যায় যা আপনার HTML নষ্ট করতে পারে।

💡 টিপস: VS Code-এ "! + Tab" চাপলেই HTML5-এর মৌলিক কাঠামো অটোমেটিক তৈরি হয়ে যায়। এটি আপনার সময় বাঁচাবে।

💡 টিপস: কোড লেখার সময় নিয়মিত "Ctrl+S" চেপে ফাইল সেভ করার অভ্যাস করুন। অপ্রত্যাশিত বন্ধ হয়ে গেলে আপনার কাজ নষ্ট হবে না।
