HTML-এর সংস্করণ (HTML Versions)

পড়তে সময়: ১০ মিনিট লেভেল: শিক্ষানবিশ

HTML সংস্করণের ধারা

HTML তার যাত্রা শুরু থেকে আজ পর্যন্ত বেশ কয়েকটি বড় সংস্করণ দেখেছে। প্রতিটি নতুন সংস্করণ আগের সংস্করণের সীমাবদ্ধতা দূর করে নতুন বৈশিষ্ট্য নিয়ে এসেছে। নিচে HTML-এর প্রধান সংস্করণগুলোর বিবরণ দেওয়া হলো।

HTML 1.0 (১৯৯১-১৯৯৫)

এটি HTML-এর প্রথম সংস্করণ, যা টিম বার্নার্স-লি তৈরি করেন। যদিও এটি আনুষ্ঠানিকভাবে মানকায়িত ছিল না, এটি ওয়েবের ভিত্তি স্থাপন করে।

বৈশিষ্ট্য:

  • মাত্র ১৮টি ট্যাগ
  • শুধুমাত্র টেক্সট এবং ছবি দেখানোর ক্ষমতা
  • হাইপারলিংক যুক্ত করার সুবিধা
  • শিরোনাম ও প্যারাগ্রাফের জন্য ট্যাগ

HTML 1.0-এর নমুনা কোড:

<title>আমার প্রথম পৃষ্ঠা</title>
<h1>স্বাগতম</h1>
<p>এটি একটি প্যারাগ্রাফ</p>
<a href="page2.html">পরবর্তী পৃষ্ঠা</a>

HTML 2.0 (১৯৯৫)

এটি ছিল HTML-এর প্রথম মানকায়িত সংস্করণ। IETF (Internet Engineering Task Force) এই সংস্করণকে আনুষ্ঠানিকভাবে অনুমোদন করে।

নতুন বৈশিষ্ট্য:

  • ফর্ম যুক্ত করা হয় (ফর্ম এলিমেন্ট)
  • টেবিল তৈরির ক্ষমতা
  • ক্লায়েন্ট-সাইড ইমেজ ম্যাপ
  • আন্তর্জাতিক অক্ষর সাপোর্ট
নতুন ট্যাগ ব্যবহার
<form> ফর্ম তৈরি
<input> ইনপুট ফিল্ড
<textarea> টেক্সট এরিয়া
<select> ড্রপডাউন তালিকা

HTML 3.2 (১৯৯৭)

W3C (World Wide Web Consortium) এই সংস্করণটি প্রকাশ করে। এটি ব্রাউজার যুদ্ধের সময়ে তৈরি হয়েছিল এবং অনেক নতুন বৈশিষ্ট্য যুক্ত হয়।

নতুন বৈশিষ্ট্য:

  • টেবিলের উন্নত সংস্করণ
  • স্ক্রিপ্ট (JavaScript) যুক্ত করার ক্ষমতা
  • স্টাইল শিট (CSS) এর প্রাথমিক সমর্থন
  • অ্যাপলেট ও ফ্ল্যাশ কন্টেন্ট দেখানোর সুবিধা
  • সেন্টার, ফন্টের মতো নতুন ট্যাগ

HTML 4.01 (১৯৯৯-২০১৪)

এটি সবচেয়ে দীর্ঘস্থায়ী HTML সংস্করণ। ১৫ বছর ধরে এটি ওয়েব ডেভেলপমেন্টের মূল ভিত্তি ছিল। তিনটি ভেরিয়েন্টে পাওয়া যেত।

তিনটি ভেরিয়েন্ট:

ভেরিয়েন্ট বর্ণনা
Strict কঠোর নিয়ম, প্রেজেন্টেশন ট্যাগ বাদ দেওয়া
Transitional পুরনো ট্যাগ ব্যবহারের অনুমতি
Frameset ফ্রেম ব্যবহারের জন্য

নতুন বৈশিষ্ট্য:

  • সিএসএস-এর পূর্ণ সমর্থন
  • অ্যাক্সেসিবিলিটি বৈশিষ্ট্য
  • আন্তর্জাতিক ভাষা সাপোর্ট
  • অবজেক্ট ট্যাগ
  • স্ক্রিপ্টিং ও ইভেন্ট হ্যান্ডলিং

XHTML 1.0 (২০০০)

XHTML ছিল HTML-এর XML-ভিত্তিক সংস্করণ। এটি HTML 4.01-কে XML-এর কঠোর নিয়মে পুনর্লিখন করে।

XHTML-এর কঠোর নিয়ম:

  • সব ট্যাগ ছোট হাতের অক্ষরে লিখতে হবে
  • সব ট্যাগ বন্ধ করতে হবে (যেমন: <br />)
  • অ্যাট্রিবিউটের মান কোটেশনের মধ্যে দিতে হবে
  • এলিমেন্ট সঠিকভাবে নেস্টেড হতে হবে

XHTML-এর নমুনা কোড:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN">
<html xmlns="http://www.w3.org/1999/xhtml">
  <head>
    <title>XHTML পৃষ্ঠা</title>
  </head>
  <body>
    <p>এটি একটি প্যারাগ্রাফ<br />নতুন লাইন</p>
    <img src="ছবি.jpg" alt="ছবি" />
  </body>
</html>

HTML5 (২০১৪-বর্তমান)

HTML5 বর্তমান সময়ের সবচেয়ে আধুনিক HTML সংস্করণ। এটি মোবাইল, ডেস্কটপ সব ধরনের ডিভাইসের জন্য তৈরি।

HTML5-এর প্রধান বৈশিষ্ট্য:

  • নতুন সিমেন্টিক ট্যাগ: <header>, <footer>, <nav>, <article>, <section>
  • মাল্টিমিডিয়া সাপোর্ট: <audio> ও <video> ট্যাগ
  • ক্যানভাস: <canvas> দিয়ে গ্রাফিক্স ও এনিমেশন
  • ফর্ম উন্নয়ন: নতুন ইনপুট টাইপ (email, date, color, range)
  • অফলাইন স্টোরেজ: লোকাল স্টোরেজ ও সেশন স্টোরেজ
  • জিওলোকেশন: ব্যবহারকারীর অবস্থান জানার সুবিধা

HTML5-এর নমুনা কোড:

<!DOCTYPE html>
<html lang="bn">
<head>
  <meta charset="UTF-8">
  <title>HTML5 পৃষ্ঠা</title>
</head>
<body>
  <header>
    <h1>আমার ওয়েবসাইট</h1>
    <nav>
      <ul>
        <li><a href="#">হোম</a></li>
        <li><a href="#">ব্লগ</a></li>
      </ul>
    </nav>
  </header>
  
  <article>
    <h2>পোস্ট শিরোনাম</h2>
    <video src="video.mp4" controls></video>
  </article>
  
  <footer>
    <p>কপিরাইট ২০২৬</p>
  </footer>
</body>
</html>

HTML 5.2 ও ভবিষ্যৎ

HTML ক্রমাগত আপডেট হচ্ছে। ২০১৭ সালে HTML 5.2 প্রকাশিত হয় এবং এখন HTML 5.3-এর কাজ চলছে।

ভবিষ্যৎ সংস্করণের সম্ভাব্য বৈশিষ্ট্য:

  • কৃত্রিম বুদ্ধিমত্তার সাথে ইন্টিগ্রেশন
  • উন্নত অ্যাক্সেসিবিলিটি
  • ভার্চুয়াল রিয়েলিটি সাপোর্ট
  • দ্রুত লোডিং ও পারফরম্যান্স

সংস্করণ তুলনামূলক চার্ট

সংস্করণ সাল মূল বৈশিষ্ট্য
HTML 1.0 ১৯৯১ মৌলিক ট্যাগ, হাইপারলিংক
HTML 2.0 ১৯৯৫ ফর্ম, টেবিল
HTML 3.2 ১৯৯৭ স্ক্রিপ্ট, স্টাইল শিট
HTML 4.01 ১৯৯৯ সিএসএস, অ্যাক্সেসিবিলিটি
XHTML 1.0 ২০০০ XML-ভিত্তিক কঠোর নিয়ম
HTML5 ২০১৪ সিমেন্টিক ট্যাগ, মাল্টিমিডিয়া

অনুশীলনী

  1. HTML-এর প্রথম মানকায়িত সংস্করণ কোনটি এবং কবে প্রকাশিত হয়?
  2. HTML 4.01-এর তিনটি ভেরিয়েন্টের নাম লেখ।
  3. XHTML-এর তিনটি কঠোর নিয়ম উদাহরণসহ লেখ।
  4. HTML5-এ যুক্ত পাঁচটি নতুন সিমেন্টিক ট্যাগের নাম লেখ।
  5. HTML 3.2 ও HTML5-এর মধ্যে প্রধান পার্থক্য কী কী?
  6. নিচের কোডটি কোন HTML সংস্করণের হতে পারে? <font color="red">টেক্সট</font>
সমাধান দেখুন
  1. HTML 2.0, ১৯৯৫ সালে
  2. Strict, Transitional, Frameset
  3. সব ট্যাগ ছোট হাতের অক্ষরে, সব ট্যাগ বন্ধ করতে হবে, অ্যাট্রিবিউটের মান কোটেশনে দিতে হবে
  4. <header>, <footer>, <nav>, <article>, <section>
  5. HTML3.2-এ প্রেজেন্টেশন ট্যাগ ছিল, HTML5 সিমেন্টিক ও মাল্টিমিডিয়া সাপোর্ট করে
  6. HTML 3.2 অথবা HTML 4.01 Transitional (কারণ <font> ট্যাগ এখন অবচিত)
