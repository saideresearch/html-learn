HTML পরিচিতি (Introduction to HTML)

পড়তে সময়: ১৫ মিনিট লেভেল: শিক্ষানবিশ

HTML কী?

HTML এর পূর্ণরূপ হলো HyperText Markup Language। এটি ওয়েবপৃষ্ঠা তৈরির জন্য ব্যবহৃত সবচেয়ে মৌলিক ভাষা। HTML-এর মাধ্যমে আমরা ওয়েবপৃষ্ঠার কাঠামো ও কন্টেন্ট নির্ধারণ করি।

মজার তথ্য:

HTML আসলে একটি প্রোগ্রামিং ভাষা নয়, এটি একটি মার্কআপ ভাষা। প্রোগ্রামিং ভাষায় লজিক ও কন্ডিশন থাকে, HTML শুধু কন্টেন্টকে সাজিয়ে দেখায়।

কেন HTML শিখবেন?

  • ওয়েব ডেভেলপমেন্টের ভিত্তি হলো HTML
  • শেখা খুব সহজ এবং দ্রুত ফল পাওয়া যায়
  • যেকোনো ওয়েবসাইট তৈরি করতে HTML জানতেই হবে
  • ফ্রিল্যান্সিং ও চাকরির বাজারে HTML-এর চাহিদা অনেক

HTML কেমন কাজ করে?

ব্রাউজার HTML কোড পড়ে এবং সেটা ভিজুয়াল ফরম্যাটে রূপান্তর করে। যেমন:

সরল উদাহরণ:

<h1>আমার প্রথম শিরোনাম</h1>
<p>এটি একটি প্যারাগ্রাফ</p>
