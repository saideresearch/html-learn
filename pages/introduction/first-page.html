প্রথম HTML পৃষ্ঠা (My First HTML Page)

পড়তে সময়: ১৫ মিনিট লেভেল: শিক্ষানবিশ হাতে-কলমে: ✅

আপনার প্রথম HTML পৃষ্ঠা তৈরি করুন

এখন আপনি শিখবেন কীভাবে একটি সম্পূর্ণ HTML পৃষ্ঠা তৈরি করতে হয়। এই টিউটোরিয়ালটি হাতে-কলমে অনুসরণ করুন। আপনার যা দরকার তা হল একটি টেক্সট এডিটর (যেমন নোটপ্যাড বা VS Code) এবং একটি ওয়েব ব্রাউজার (যেমন Chrome বা Firefox)।

শুরু করার আগে মনে রাখবেন:

  • HTML ফাইল সবসময় .html এক্সটেনশন দিয়ে সংরক্ষণ করতে হবে
  • ফাইলের নাম ছোট হাতের অক্ষরে এবং স্পেস ছাড়া দেওয়া ভালো (যেমন: first-page.html)
  • কোড লেখার সময় প্রতিটি ট্যাগ সঠিকভাবে বন্ধ করছেন কিনা দেখুন

ধাপ ১: একটি নতুন ফাইল তৈরি করুন

VS Code-এ:

  1. VS Code খুলুন
  2. File → New File (Ctrl+N) ক্লিক করুন
  3. File → Save As (Ctrl+Shift+S) ক্লিক করুন
  4. ফাইলের নাম দিন: first-page.html
  5. Save ক্লিক করুন

নোটপ্যাডে:

  1. নোটপ্যাড খুলুন
  2. File → Save As ক্লিক করুন
  3. Save as type: "All Files (*.*)" নির্বাচন করুন
  4. ফাইলের নাম দিন: first-page.html
  5. Save ক্লিক করুন

💡 টিপস: আপনার ডেস্কটপে "HTML Projects" নামে একটি ফোল্ডার তৈরি করে সব HTML ফাইল সেখানে সংরক্ষণ করুন। এতে পরে খুঁজে পেতে সুবিধা হবে।

ধাপ ২: মৌলিক HTML কাঠামো লেখা

প্রত্যেকটি HTML পৃষ্ঠার একটি নির্দিষ্ট কাঠামো থাকে। নিচের কোডটি আপনার ফাইলে কপি করে পেস্ট করুন:

<!DOCTYPE html>
<html>
<head>
  <title>আমার প্রথম ওয়েবপৃষ্ঠা</title>
</head>
<body>

</body>
</html>

কোডের ব্যাখ্যা:

ট্যাগ কাজ
<!DOCTYPE html> ব্রাউজারকে জানায় যে এটি HTML5 ডকুমেন্ট
<html> পুরো HTML ডকুমেন্টের মূল এলিমেন্ট (রুট এলিমেন্ট)
<head> পৃষ্ঠার মেটাডাটা (যেমন টাইটেল, স্টাইল, স্ক্রিপ্ট) রাখার জায়গা
<title> ব্রাউজার ট্যাবে কী দেখাবে তা নির্ধারণ করে
<body> পৃষ্ঠায় যা দেখা যাবে (কন্টেন্ট) তা এখানে লেখা হয়

ধাপ ৩: কন্টেন্ট যোগ করা

এখন <body> ট্যাগের ভিতরে কিছু কন্টেন্ট যোগ করি। নিচের কোডটি আপনার <body> ট্যাগের ভিতরে বসান:

<h1>আমার প্রথম HTML পৃষ্ঠায় স্বাগতম!</h1>
<p>এই পৃষ্ঠাটি আমি HTML শিখি ওয়েবসাইটের টিউটোরিয়াল অনুসরণ করে তৈরি করেছি।</p>
<p>HTML শেখা খুবই মজার এবং সহজ। আপনি যদি নিয়মিত অনুশীলন করেন, তাহলে খুব দ্রুত ওয়েব ডেভেলপার হতে পারবেন।</p>
<h2>আজকে আমি যা শিখলাম:</h2>
<ul>
  <li>HTML ফাইল তৈরি করা</li>
  <li>মৌলিক HTML কাঠামো</li>
  <li>শিরোনাম ও প্যারাগ্রাফ যোগ করা</li>
  <li>তালিকা তৈরি করা</li>
</ul>

ধাপ ৪: সম্পূর্ণ কোড

উপরের সবগুলো একসাথে করলে আপনার সম্পূর্ণ HTML ফাইলটি নিচের মত দেখাবে:

<!DOCTYPE html>
<html>
<head>
  <title>আমার প্রথম ওয়েবপৃষ্ঠা</title>
</head>
<body>
  <h1>আমার প্রথম HTML পৃষ্ঠায় স্বাগতম!</h1>
  <p>এই পৃষ্ঠাটি আমি HTML শিখি ওয়েবসাইটের টিউটোরিয়াল অনুসরণ করে তৈরি করেছি।</p>
  <p>HTML শেখা খুবই মজার এবং সহজ। আপনি যদি নিয়মিত অনুশীলন করেন, তাহলে খুব দ্রুত ওয়েব ডেভেলপার হতে পারবেন।</p>
  <h2>আজকে আমি যা শিখলাম:</h2>
  <ul>
    <li>HTML ফাইল তৈরি করা</li>
    <li>মৌলিক HTML কাঠামো</li>
    <li>শিরোনাম ও প্যারাগ্রাফ যোগ করা</li>
    <li>তালিকা তৈরি করা</li>
  </ul>
</body>
</html>

এখন আপনার ফাইলটি সেভ করুন (Ctrl+S)।

ধাপ ৫: ব্রাউজারে দেখুন

এখন আপনার তৈরি HTML পৃষ্ঠাটি ব্রাউজারে খুলে দেখুন।

পদ্ধতি:

  1. আপনি যেখানে first-page.html ফাইলটি সেভ করেছেন, সেই ফোল্ডারে যান
  2. first-page.html ফাইলটিতে ডাবল-ক্লিক করুন
  3. ফাইলটি আপনার ডিফল্ট ব্রাউজারে খুলে যাবে

VS Code-এ লাইভ প্রিভিউ (ঐচ্ছিক):

আপনি যদি VS Code ব্যবহার করেন এবং Live Server এক্সটেনশন ইনস্টল করে থাকেন, তাহলে:

  1. HTML ফাইলটিতে রাইট-ক্লিক করুন
  2. "Open with Live Server" নির্বাচন করুন
  3. আপনার ডিফল্ট ব্রাউজারে পৃষ্ঠাটি খুলবে এবং আপনি কোড পরিবর্তন করলেই সাথে সাথে ব্রাউজার আপডেট হবে

ব্রাউজারে ফলাফল

আপনার ব্রাউজারে নিচের মত একটি পৃষ্ঠা দেখা যাবে:

সাধারণ ভুল ও সমাধান

সমস্যা কারণ সমাধান
ব্রাউজারে HTML কোড দেখা যাচ্ছে, পৃষ্ঠা না ফাইল .html এক্সটেনশন দিয়ে সেভ করা হয়নি .html এক্সটেনশন দিয়ে ফাইলটি সেভ করুন
ট্যাগ ব্রাউজারে দেখা যাচ্ছে ট্যাগ ঠিকমত বন্ধ করা হয়নি বা < ও > এর পরিবর্তে < এবং > ব্যবহার করা হয়েছে ট্যাগ সঠিকভাবে বন্ধ করুন এবং < ও > ব্যবহার করুন
ছবি দেখা যাচ্ছে না ছবির লিংক ভুল বা ছবির ফাইল সঠিক জায়গায় নেই সঠিক URL বা ফাইল পাথ ব্যবহার করুন
বাংলা টেক্সট ঠিকমত দেখাচ্ছে না meta charset ট্যাগ নেই <head> ট্যাগের ভিতরে <meta charset="UTF-8"> যোগ করুন

বাংলা ভাষায় HTML পৃষ্ঠা

বাংলা ভাষায় HTML পৃষ্ঠা তৈরি করতে <meta charset="UTF-8"> ট্যাগটি <head> ট্যাগের ভিতরে যোগ করতে হবে। এছাড়া <html> ট্যাগে lang="bn" অ্যাট্রিবিউট যোগ করলে ভালো হয়।

<!DOCTYPE html>
<html lang="bn">
<head>
  <meta charset="UTF-8">
  <title>বাংলায় HTML পৃষ্ঠা</title>
</head>
<body>
  <h1>বাংলায় স্বাগতম!</h1>
  <p>এটি একটি বাংলা HTML পৃষ্ঠা। এখন বাংলা ভাষা সঠিকভাবে দেখাবে।</p>
</body>
</html>

পরবর্তী ধাপ: আপনার পৃষ্ঠাকে অনলাইনে প্রকাশ করা

আপনার তৈরি HTML পৃষ্ঠাটি যদি অনলাইনে সবাইকে দেখাতে চান, তাহলে নিচের পদ্ধতি অনুসরণ করতে পারেন:

বিনামূল্যে হোস্টিং পরিষেবা:

  • GitHub Pages: আপনার HTML ফাইল GitHub-এ আপলোড করে ফ্রিতে ওয়েবসাইট হোস্ট করতে পারেন
  • Netlify: ড্র্যাগ অ্যান্ড ড্রপ করে HTML ফাইল আপলোড দিলেই লাইভ লিংক পেয়ে যাবেন
  • Vercel: খুব সহজে HTML সাইট হোস্ট করা যায়
  • InfinityFree: ১০০% ফ্রি হোস্টিং, কোনো বিজ্ঞাপন নেই

💡 টিপস: GitHub Pages ব্যবহার করা সবচেয়ে ভালো কারণ এটি ফ্রি এবং আপনার কোড সংরক্ষিত থাকে। শুধু একটি GitHub অ্যাকাউন্ট খুলুন, একটি রিপোজিটরি তৈরি করুন, সেখানে আপনার HTML ফাইল আপলোড করুন এবং সেটিংস থেকে GitHub Pages চালু করুন।

অনুশীলনী

নিচের অনুশীলনীগুলো নিজে করার চেষ্টা করুন। এগুলো আপনার শেখা জিনিসগুলো মনে রাখতে সাহায্য করবে।

  1. আপনার নাম, ঠিকানা এবং প্রিয় খাবার নিয়ে একটি HTML পৃষ্ঠা তৈরি করুন।
  2. একটি HTML পৃষ্ঠা তৈরি করুন যেখানে আপনার প্রিয় তিনটি ওয়েবসাইটের লিংক থাকবে। লিংকগুলো নতুন ট্যাবে খুলবে।
  3. একটি HTML পৃষ্ঠা তৈরি করুন যেখানে আপনার প্রিয় একটি ছবি থাকবে এবং ছবির নিচে ছবিটি সম্পর্কে বর্ণনা থাকবে।
  4. উপরের আপডেটেড কোডটি কপি করে আপনার এডিটরে পেস্ট করুন এবং ব্রাউজারে ফলাফল দেখুন। তারপর কোডের প্রতিটি অংশ পরিবর্তন করে দেখুন কী হয়।
  5. আপনার তৈরি প্রথম HTML পৃষ্ঠায় একটি অনুচ্ছেদ যোগ করুন যেখানে আপনি HTML সম্পর্কে আপনার মতামত লিখবেন।
