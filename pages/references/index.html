HTML রেফারেন্স (HTML Reference)

পড়তে সময়: দ্রুত রেফারেন্স লেভেল: সকল স্তরের জন্য

HTML রেফারেন্স কী?

HTML রেফারেন্স হলো HTML ডেভেলপমেন্টের সময় দ্রুত তথ্য পাওয়ার জন্য একটি কম্প্যাক্ট এবং সংগঠিত নির্দেশিকা। এখানে আপনি HTML ট্যাগ, অ্যাট্রিবিউট, গ্লোবাল অ্যাট্রিবিউট, ইভেন্ট অ্যাট্রিবিউট, ইনপুট টাইপ, ক্যারেক্টার এনটিটি, কালার কোড, মাইম টাইপ এবং একটি সম্পূর্ণ চিট শিট পাবেন। এটি শিক্ষানবিস থেকে শুরু করে অভিজ্ঞ ডেভেলপারদের জন্য একটি দ্রুতগতির রেফারেন্স গাইড হিসেবে কাজ করবে। কোনো ট্যাগ বা অ্যাট্রিবিউটের নাম মনে না পড়লে, বা সঠিক সিনট্যাক্স জানতে চাইলে এই সেকশনটি ব্যবহার করুন।

রেফারেন্স ব্যবহারের উপায়:

  • কোনো ট্যাগের সিনট্যাক্স দ্রুত দেখতে এই পৃষ্ঠাগুলো ব্যবহার করুন
  • নতুন ট্যাগ বা অ্যাট্রিবিউট সম্পর্কে জানতে ব্রাউজ করুন
  • চিট শিট প্রিন্ট করে ডেস্কে রাখুন দ্রুত রেফারেন্সের জন্য
  • ক্যারেক্টার এনটিটি এবং কালার কোড দ্রুত খুঁজে পেতে ব্যবহার করুন

প্রয়োজনীয় এনটিটি

  • &lt; → < (less than)
  • &gt; → > (greater than)
  • &amp; → & (ampersand)
  • &quot; → " (double quote)
  • &apos; → ' (apostrophe)
  • &nbsp; →   (non-breaking space)

কারেন্সি সিম্বল

  • &copy; → © (copyright)
  • &reg; → ® (registered)
  • &trade; → ™ (trademark)
  • &euro; → € (euro)
  • &pound; → £ (pound)
  • &yen; → ¥ (yen)

অ্যারো সিম্বল

  • &larr; → ← (left arrow)
  • &rarr; → → (right arrow)
  • &uarr; → ↑ (up arrow)
  • &darr; → ↓ (down arrow)
  • &harr; → ↔ (left-right arrow)
