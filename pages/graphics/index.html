গ্রাফিক্স (HTML Graphics)

পড়তে সময়: ৫৫ মিনিট লেভেল: মধ্যবর্তী থেকে উন্নত

HTML গ্রাফিক্স কী?

HTML-এ গ্রাফিক্স তৈরির জন্য দুটি শক্তিশালী টুল রয়েছে: Canvas এবং SVG। <canvas> এলিমেন্ট জাভাস্ক্রিপ্ট ব্যবহার করে পিক্সেল-ভিত্তিক গ্রাফিক্স, অ্যানিমেশন, এবং গেম ডেভেলপমেন্টের জন্য উপযোগী। অন্যদিকে SVG (Scalable Vector Graphics) XML-ভিত্তিক ভেক্টর গ্রাফিক্স যা স্কেলেবল, রেসপনসিভ এবং ডকুমেন্ট অবজেক্ট মডেল (DOM) এর সাথে ইন্টিগ্রেটেড। এই অধ্যায়ে আমরা Canvas এবং SVG-এর সকল গুরুত্বপূর্ণ বিষয় বিস্তারিত শিখব।

শিখবেন যা:

  • Canvas: বেসিক ড্রয়িং, শেপ, রং, গ্রেডিয়েন্ট, টেক্সট, ছবি, অ্যানিমেশন
  • SVG: ভেক্টর গ্রাফিক্স, শেপ, পাথ, টেক্সট, ইন্টারঅ্যাকটিভিটি
  • Canvas vs SVG: কখন কোনটি ব্যবহার করবেন
  • জাভাস্ক্রিপ্ট দিয়ে ইন্টারঅ্যাকটিভ গ্রাফিক্স তৈরি

Canvas বেসিক (Canvas Basics)

<canvas> এলিমেন্ট হলো একটি ড্রয়িং এরিয়া যা জাভাস্ক্রিপ্ট দিয়ে নিয়ন্ত্রণ করা হয়। এটি 2D গ্রাফিক্স, অ্যানিমেশন, এবং গেম ডেভেলপমেন্টের জন্য আদর্শ।

Canvas স্ট্রাকচার:

<canvas id="myCanvas" width="400" height="200" style="border:1px solid #ddd;">
  আপনার ব্রাউজার canvas সাপোর্ট করে না।
</canvas>

<script>
  const canvas = document.getElementById('myCanvas');
  const ctx = canvas.getContext('2d');
  // ড্রয়িং কোড এখানে
  ctx.fillStyle = '#4CAF50';
  ctx.fillRect(20, 20, 150, 100);
</script>

Canvas এর মূল বৈশিষ্ট্য:

  • পিক্সেল-ভিত্তিক (রাস্টার গ্রাফিক্স)
  • জাভাস্ক্রিপ্ট দিয়ে রিয়েল-টাইম ড্রয়িং
  • অ্যানিমেশন ও গেম ডেভেলপমেন্টের জন্য উপযোগী
  • হাই-রেজোলিউশনে দ্রুত পারফরম্যান্স
  • ইমেজ ডাটা ম্যানিপুলেশন সম্ভব

Canvas ড্রয়িং (Drawing Shapes)

Canvas-এ আয়তক্ষেত্র, বৃত্ত, লাইন, এবং পাথ ব্যবহার করে বিভিন্ন শেপ আঁকা যায়।

বিভিন্ন শেপ ড্রয়িং:

// আয়তক্ষেত্র
ctx.fillRect(x, y, width, height);
ctx.strokeRect(x, y, width, height);

// বৃত্ত
ctx.beginPath();
ctx.arc(x, y, radius, startAngle, endAngle);
ctx.fill();

// লাইন
ctx.beginPath();
ctx.moveTo(x1, y1);
ctx.lineTo(x2, y2);
ctx.stroke();

Canvas রং ও গ্রেডিয়েন্ট (Colors & Gradients)

Canvas-এ সলিড রং, লিনিয়ার গ্রেডিয়েন্ট, এবং রেডিয়াল গ্রেডিয়েন্ট ব্যবহার করে আকর্ষণীয় ডিজাইন তৈরি করা যায়।

গ্রেডিয়েন্ট উদাহরণ:

// লিনিয়ার গ্রেডিয়েন্ট
const gradient = ctx.createLinearGradient(0, 0, 200, 0);
gradient.addColorStop(0, 'red');
gradient.addColorStop(1, 'blue');
ctx.fillStyle = gradient;
ctx.fillRect(0, 0, 200, 100);

// রেডিয়াল গ্রেডিয়েন্ট
const radialGrad = ctx.createRadialGradient(100, 50, 10, 100, 50, 40);
radialGrad.addColorStop(0, '#ff6b6b');
radialGrad.addColorStop(1, '#4ecdc4');
ctx.fillStyle = radialGrad;
ctx.fillRect(0, 0, 200, 100);

Canvas টেক্সট (Text on Canvas)

Canvas-এ fillText() এবং strokeText() মেথড ব্যবহার করে টেক্সট যোগ করা যায়। ফন্ট, সাইজ, এবং অ্যালাইনমেন্ট কাস্টমাইজ করা যায়।

Canvas টেক্সট উদাহরণ:

ctx.font = '30px Arial';
ctx.fillStyle = '#333';
ctx.fillText('HTML Canvas', 50, 50);

ctx.font = 'bold 20px "Segoe UI"';
ctx.strokeStyle = '#2196F3';
ctx.strokeText('বাংলা টেক্সট', 50, 100);

Canvas ছবি (Images on Canvas)

drawImage() মেথড ব্যবহার করে Canvas-এ ছবি আঁকা যায়। ইমেজ ক্রপ, স্কেল, এবং ম্যানিপুলেট করা সম্ভব।

Canvas-এ ছবি আঁকা:

const img = new Image();
img.onload = function() {
  ctx.drawImage(img, 0, 0, 200, 150);
};
img.src = 'image.jpg';

Canvas এনিমেশন (Animation)

requestAnimationFrame() ব্যবহার করে Canvas-এ মসৃণ অ্যানিমেশন তৈরি করা যায়। ফ্রেম-বাই-ফ্রেম ড্রয়িং এবং মুভিং অবজেক্ট তৈরি সম্ভব।

সরল এনিমেশন উদাহরণ:

let x = 0;
function animate() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  ctx.fillStyle = '#4CAF50';
  ctx.fillRect(x, 50, 50, 50);
  x += 2;
  if(x > canvas.width) x = -50;
  requestAnimationFrame(animate);
}
animate();
🟢 চলমান বক্স এনিমেশন

SVG বেসিক (SVG Basics)

SVG (Scalable Vector Graphics) হলো XML-ভিত্তিক ভেক্টর গ্রাফিক্স ফরম্যাট। এটি স্কেলেবল, রেসপনসিভ, এবং CSS ও JavaScript দিয়ে সহজে নিয়ন্ত্রণ করা যায়। SVG এলিমেন্ট সরাসরি DOM-এর অংশ হয়।

SVG উদাহরণ:

<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <circle cx="100" cy="100" r="80" fill="#4CAF50" />
  <rect x="50" y="50" width="100" height="100" fill="none" stroke="white" stroke-width="3" />
  <text x="100" y="110" text-anchor="middle" fill="white" font-size="20">SVG</text>
</svg>

SVG এর মূল বৈশিষ্ট্য:

  • ভেক্টর-ভিত্তিক (স্কেলেবল, পিক্সেলেশন ছাড়া)
  • XML ফরম্যাট, DOM-এর সাথে ইন্টিগ্রেটেড
  • CSS দিয়ে স্টাইলিং ও JavaScript দিয়ে ইন্টারঅ্যাকশন সম্ভব
  • রেসপনসিভ ডিজাইনের জন্য আদর্শ
  • অ্যাক্সেসিবিলিটি সাপোর্ট করে

SVG শেপ (Shapes)

SVG-তে বিভিন্ন শেপ যেমন আয়তক্ষেত্র, বৃত্ত, উপবৃত্ত, লাইন, পলিলাইন, পলিগন ইত্যাদি তৈরি করা যায়।

বিভিন্ন SVG শেপ:

<rect x="10" y="10" width="80" height="60" fill="#FF5722" />
<circle cx="150" cy="40" r="30" fill="#2196F3" />
<ellipse cx="250" cy="40" rx="35" ry="20" fill="#FFC107" />
<polygon points="350,10 370,35 350,60 330,35" fill="#9C27B0" />

SVG পাথ (Paths)

<path> এলিমেন্ট SVG-তে সবচেয়ে শক্তিশালী টুল। এটি দিয়ে জটিল আকার, কার্ভ, এবং কাস্টম শেপ তৈরি করা যায়।

SVG পাথ উদাহরণ:

<path d="M 10 80 Q 95 10 180 80" stroke="blue" fill="none" stroke-width="3"/>
<path d="M 10,100 L 50,20 L 90,100 Z" fill="orange" stroke="brown" stroke-width="2"/>

SVG টেক্সট (Text in SVG)

SVG-তে <text> এলিমেন্ট ব্যবহার করে টেক্সট যোগ করা যায়। টেক্সট পাথে ফোলো করতে পারে, অ্যানিমেটেড হতে পারে, এবং CSS দিয়ে স্টাইল করা যায়।

SVG টেক্সট উদাহরণ:

<svg>
  <text x="20" y="30" fill="red" font-size="20">Hello SVG!</text>
  <text x="20" y="60" fill="green" font-weight="bold">বাংলা টেক্সট</text>
</svg>

Canvas vs SVG - কখন কোনটি ব্যবহার করবেন?

Canvas এবং SVG-এর মধ্যে নির্বাচন প্রকল্পের প্রয়োজন অনুযায়ী করা উচিত। নিচে তুলনামূলক বিশ্লেষণ দেওয়া হলো:

বৈশিষ্ট্যCanvasSVG
গ্রাফিক্স টাইপরাস্টার (পিক্সেল-ভিত্তিক)ভেক্টর (স্কেলেবল)
পারফরম্যান্সজটিল অ্যানিমেশনে ভালোসহজ গ্রাফিক্সে ভালো, বড় ডাটাতে ধীর
DOM ইন্টিগ্রেশননা (শুধু JavaScript)হ্যাঁ (প্রত্যেক এলিমেন্ট DOM নোড)
ইভেন্ট হ্যান্ডলিংহ্যাঁ (কোঅর্ডিনেট ভিত্তিক)হ্যাঁ (প্রত্যেক এলিমেন্টে সরাসরি)
স্কেলিংপিক্সেলেশন হয়যেকোনো সাইজে পরিষ্কার
উপযোগিতাগেম, অ্যানিমেশন, ইমেজ ম্যানিপুলেশনআইকন, লোগো, চার্ট, রেসপনসিভ গ্রাফিক্স

সিদ্ধান্ত নেওয়ার টিপস:

  • Canvas ব্যবহার করুন: যখন রিয়েল-টাইম অ্যানিমেশন, গেম, পিক্সেল ম্যানিপুলেশন, বা উচ্চ ফ্রেম রেট প্রয়োজন
  • SVG ব্যবহার করুন: যখন স্কেলেবল গ্রাফিক্স, আইকন, লোগো, চার্ট, বা DOM-ভিত্তিক ইন্টারঅ্যাকশন প্রয়োজন
  • সংযুক্ত ব্যবহার: অনেক বড় প্রকল্পে Canvas এবং SVG একসাথে ব্যবহার করা যায়
