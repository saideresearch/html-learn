HTML অনুশীলনী (HTML Exercises)

অনুশীলন সংখ্যা: ৫০+ লেভেল: শিক্ষানবিশ থেকে অ্যাডভান্সড

অনুশীলন কেন গুরুত্বপূর্ণ?

HTML শেখার সবচেয়ে কার্যকরী পদ্ধতি হলো হাতে-কলমে অনুশীলন করা। তত্ত্বীয় জ্ঞান শুধুমাত্র ধারণা দেয়, কিন্তু প্রকৃত দক্ষতা আসে প্র্যাকটিক্যাল কাজের মাধ্যমে। এই অনুশীলনীগুলো ধাপে ধাপে সাজানো হয়েছে—সহজ থেকে জটিল, মৌলিক HTML ট্যাগ থেকে শুরু করে অ্যাডভান্সড কনসেপ্ট পর্যন্ত। প্রতিটি অনুশীলন শেষে আপনি নতুন কিছু শিখবেন এবং আপনার আত্মবিশ্বাস বাড়বে।

অনুশীলন করার পদ্ধতি:

  • প্রথমে নিজে নিজে সমাধান করার চেষ্টা করুন
  • কোড লিখে ব্রাউজারে রিয়েল-টাইম ফলাফল দেখুন
  • সমাধান দেখার আগে কমপক্ষে ১৫-২০ মিনিট চেষ্টা করুন
  • ভুল হলে বারবার চেষ্টা করুন—ভুল থেকেই শেখা হয়
  • সমাধান হয়ে গেলে কোড ভ্যালিডেট করুন (validator.w3.org)

শিক্ষানবিশ অনুশীলনী (প্রথম ধাপ)

যারা HTML শুরু করছেন, তাদের জন্য মৌলিক অনুশীলন। নিচের কিছু নমুনা অনুশীলন দেওয়া হলো:

অনুশীলন ১: আমার প্রথম ওয়েবপৃষ্ঠা

একটি সম্পূর্ণ HTML ডকুমেন্ট তৈরি করুন যাতে থাকবে:

  • সঠিক DOCTYPE এবং html ট্যাগ
  • একটি শিরোনাম (title) "আমার প্রথম পৃষ্ঠা"
  • একটি h1 হেডিং "স্বাগতম"
  • একটি প্যারাগ্রাফ "এটি আমার প্রথম HTML পৃষ্ঠা"
সমাধান দেখুন 
<!DOCTYPE html>
<html lang="bn">
<head>
  <meta charset="UTF-8">
  <title>আমার প্রথম পৃষ্ঠা</title>
</head>
<body>
  <h1>স্বাগতম</h1>
  <p>এটি আমার প্রথম HTML পৃষ্ঠা</p>
</body>
</html>

একটি আনঅর্ডারড লিস্ট তৈরি করুন যাতে আপনার প্রিয় ৫টি খাবারের নাম থাকবে।

সমাধান দেখুন 
<ul>
  <li>>বিরিয়ানি</li>
  <li>>ফুচকা</li>
  <li>>চিকেন রোস্ট</li>
  <li>>কাচ্চি</li>
  <li>>মোরগ পোলাও</li>
</ul>

অনুশীলন ৩: লিংক ও ইমেজ

একটি পৃষ্ঠা তৈরি করুন যাতে থাকবে:

  • একটি ছবি (অনলাইন থেকে যেকোনো ছবির URL ব্যবহার করুন)
  • ছবিতে alt টেক্সট দিন
  • একটি লিংক যা https://www.google.com-এ যাবে এবং নতুন ট্যাবে খুলবে

অনুশীলন ১: টেবল তৈরি

একটি টেবল তৈরি করুন যাতে ৩টি কলাম থাকবে: নাম, বয়স, শহর। কমপক্ষে ৩টি সারি ডাটা যোগ করুন। টেবলে হেডার ব্যবহার করুন।

কোডিং চ্যালেঞ্জ (নমুনা)

প্রকৃত ওয়েব ডেভেলপমেন্ট পরিস্থিতি ভিত্তিক চ্যালেঞ্জ।

চ্যালেঞ্জ ১: নেভিগেশন মেনু তৈরি করুন

একটি নেভিগেশন মেনু তৈরি করুন যাতে ৪টি লিংক থাকবে: হোম, ব্লগ, সার্ভিসেস, কন্টাক্ট। লিংকগুলো সঠিকভাবে সাজান এবং মেনুটি একটি <nav> এলিমেন্টের ভিতরে রাখুন।

সমাধান দেখুন 
<nav>
  <ul>
    <li><a href="#">হোম</a></li>
    <li><a href="#">ব্লগ</a></li>
    <li><a href="#">সার্ভিসেস</a></li>
    <li><a href="#">কন্টাক্ট</a></li>
  </ul>
</nav>

চ্যালেঞ্জ ২: প্রোডাক্ট কার্ড তৈরি করুন

একটি পণ্যের কার্ড তৈরি করুন যাতে থাকবে: পণ্যের ছবি (placeholder), পণ্যের নাম, মূল্য, এবং একটি "কিনুন" বাটন। ছবির জন্য placeholder হিসেবে LoremFlick বা যেকোনো ডামি ইমেজ URL ব্যবহার করুন।

অনুশীলন করার টিপস

সফলভাবে অনুশীলন করার কৌশল:

  • প্রতিদিন অনুশীলন করুন: নিয়মিত ৩০-৪৫ মিনিট অনুশীলন করলে দক্ষতা দ্রুত বৃদ্ধি পায়
  • কোড নিজে টাইপ করুন: কপি-পেস্ট এড়িয়ে চলুন, নিজে টাইপ করলে ভালোভাবে মনে থাকে
  • ভুল থেকে শিখুন: ভুল হলে বারবার চেষ্টা করুন, ভুল করাই শেখার অংশ
  • ব্রাউজারে পরীক্ষা করুন: কোড লেখার পর ব্রাউজারে রান করে ফলাফল দেখুন
  • ডেভেলপার টুলস ব্যবহার করুন: Chrome DevTools দিয়ে কোড ডিবাগ করুন
  • অন্যের কোড পড়ুন: GitHub-এ অন্যের HTML কোড দেখে নতুন জিনিস শিখুন
  • প্রজেক্ট তৈরি করুন: অনুশীলন শেষে ছোট ছোট প্রজেক্ট তৈরি করুন
