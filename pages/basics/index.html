HTML বেসিক (HTML Basics)

পড়তে সময়: ২০ মিনিট লেভেল: শিক্ষানবিশ

HTML বেসিক কী কী?

HTML-এর মৌলিক বিষয়গুলো শিখলে আপনি যেকোনো ওয়েবপৃষ্ঠার কাঠামো বুঝতে পারবেন এবং নিজে ছোটখাটো ওয়েবপৃষ্ঠা তৈরি করতে পারবেন। এই অধ্যায়ে আমরা নিম্নলিখিত বিষয়গুলো শিখব:

  • HTML ডকুমেন্টের মৌলিক কাঠামো
  • হেডিং ট্যাগ (h1-h6)
  • প্যারাগ্রাফ ট্যাগ (p)
  • টেক্সট ফরম্যাটিং (strong, em, mark, small)
  • HTML কমেন্ট
  • কালার ব্যবহার
  • অ্যাট্রিবিউট

HTML ডকুমেন্টের মৌলিক কাঠামো

প্রতিটি HTML ডকুমেন্টের একটি নির্দিষ্ট কাঠামো থাকে যা ব্রাউজার বুঝতে পারে। নিচে একটি সাধারণ HTML ডকুমেন্টের উদাহরণ দেওয়া হলো:

সরল উদাহরণ:

<!DOCTYPE html>
<html lang="bn">
<head>
  <meta charset="UTF-8">
  <title>আমার প্রথম পৃষ্ঠা</title>
</head>
<body>
  <h1>স্বাগতম</h1>
  <p>এটি আমার প্রথম HTML পৃষ্ঠা।</p>
</body>
</html>

স্বাগতম

এটি আমার প্রথম HTML পৃষ্ঠা।

মনে রাখবেন:

সব HTML ডকুমেন্ট <!DOCTYPE html> দিয়ে শুরু হয়। এটি ব্রাউজারকে বলে যে এটি HTML5 ডকুমেন্ট।

টেক্সট ফরম্যাটিং উদাহরণ

HTML-এ টেক্সট ফরম্যাটিং করার জন্য বিভিন্ন ট্যাগ ব্যবহার করা হয়:

টেক্সট ফরম্যাটিং:

<strong>বোল্ড টেক্সট</strong>
<em>ইটালিক টেক্সট</em>
<mark>হাইলাইটেড টেক্সট</mark>
<small>ছোট টেক্সট</small>
<del>ডিলিটেড টেক্সট</del>
<ins>ইনসার্টেড টেক্সট</ins>
