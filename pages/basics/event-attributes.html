ইভেন্ট অ্যাট্রিবিউট (Event Attributes)

পড়তে সময়: ২০ মিনিট লেভেল: মাঝারি

ইভেন্ট অ্যাট্রিবিউট কী?

ইভেন্ট অ্যাট্রিবিউট হল HTML-এর বিশেষ অ্যাট্রিবিউট যা ব্রাউজারের বিভিন্ন ইভেন্টের (যেমন ক্লিক, মাউস মুভ, কী প্রেস) সাথে জাভাস্ক্রিপ্ট কোড সংযুক্ত করে। এগুলি ব্যবহার করে ওয়েবপৃষ্ঠাকে ইন্টারঅ্যাকটিভ এবং ডায়নামিক করা যায়। যখন কোনো ব্যবহারকারী একটি এলিমেন্টের সাথে ইন্টারঅ্যাক্ট করে, তখন সংশ্লিষ্ট ইভেন্ট অ্যাট্রিবিউটের জাভাস্ক্রিপ্ট কোড এক্সিকিউট হয়।

ইভেন্ট অ্যাট্রিবিউট সব HTML এলিমেন্টের সাথে ব্যবহার করা যায় এবং এগুলি ওয়েব অ্যাপ্লিকেশন ডেভেলপমেন্টের ভিত্তি।

💡 গুরুত্বপূর্ণ: ইভেন্ট অ্যাট্রিবিউটের মান হল জাভাস্ক্রিপ্ট কোড। আধুনিক ওয়েব ডেভেলপমেন্টে ইভেন্ট হ্যান্ডলিংয়ের জন্য addEventListener() পদ্ধতি বেশি ব্যবহৃত হলেও, ইভেন্ট অ্যাট্রিবিউট দ্রুত পরীক্ষা-নিরীক্ষার জন্য খুবই কার্যকরী।

মাউস ইভেন্ট অ্যাট্রিবিউট

মাউস ইভেন্ট অ্যাট্রিবিউট ব্যবহারকারীর মাউসের বিভিন্ন動作ের সাথে সম্পর্কিত। নিচে গুরুত্বপূর্ণ মাউস ইভেন্টগুলোর তালিকা দেওয়া হলো:

অ্যাট্রিবিউট বিবরণ কখন ট্রিগার হয়
onclick এলিমেন্টে ক্লিক করলে মাউস ক্লিকের সময়
ondblclick এলিমেন্টে ডাবল ক্লিক করলে দুবার দ্রুত ক্লিক করলে
onmouseenter মাউস এলিমেন্টে প্রবেশ করলে মাউস এলিমেন্টের ভিতরে আসলে
onmouseleave মাউস এলিমেন্ট ছেড়ে গেলে মাউস এলিমেন্টের বাইরে গেলে
onmouseover মাউস এলিমেন্টের উপর গেলে মাউস এলিমেন্টের উপর এলে (বাবল হয়)
onmouseout মাউস এলিমেন্ট থেকে বেরিয়ে গেলে মাউস বেরিয়ে গেলে (বাবল হয়)
onmousemove মাউস এলিমেন্টের উপর মুভ করলে মাউস নাড়ানোর সময়
oncontextmenu রাইট ক্লিক করলে কনটেক্সট মেনু খোলার সময়

উদাহরণ: মাউস ইভেন্ট

<button onclick="alert('ক্লিক করা হয়েছে!')">
  ক্লিক করুন
</button>

<div onmouseenter="this.style.backgroundColor='lightblue'" 
     onmouseleave="this.style.backgroundColor='lightgray'"
     style="width:200px; height:100px; background:lightgray; text-align:center; line-height:100px;">
  মাউস আনুন
</div>

<p ondblclick="alert('ডাবল ক্লিক!')">
  এই টেক্সটে ডাবল ক্লিক করুন
</p>

কীবোর্ড ইভেন্ট অ্যাট্রিবিউট

কীবোর্ড ইভেন্ট অ্যাট্রিবিউট ব্যবহারকারীর কীবোর্ডের বিভিন্ন动作ের সাথে সম্পর্কিত।

অ্যাট্রিবিউট বিবরণ কখন ট্রিগার হয়
onkeydown কী চাপা হলে কী চাপার মুহূর্তে
onkeypress কী চেপে ধরা হলে কী চাপা অবস্থায় (ডিপ্রিকেটেড)
onkeyup কী ছেড়ে দিলে কী ছেড়ে দেওয়ার সময়

উদাহরণ: কীবোর্ড ইভেন্ট

<input type="text" 
       onkeydown="console.log('Key down: ' + event.key)"
       onkeyup="this.style.backgroundColor='lightgreen'; setTimeout(()=>this.style.backgroundColor='',500)"
       placeholder="কোনো কী চাপুন">

<p>আপনি চাপছেন: <span id="keyDisplay">-</span></p>

<script>
document.querySelector('input').onkeypress = function(e) {
  document.getElementById('keyDisplay').innerText = e.key;
}
</script>

ফর্ম ইভেন্ট অ্যাট্রিবিউট

ফর্ম ইভেন্ট অ্যাট্রিবিউট ফর্ম এলিমেন্টের বিভিন্ন অবস্থার সাথে সম্পর্কিত।

অ্যাট্রিবিউট বিবরণ কখন ট্রিগার হয়
onfocus এলিমেন্ট ফোকাস পেলে ইনপুট ফিল্ডে ক্লিক করলে
onblur এলিমেন্ট ফোকাস হারালে ইনপুট ফিল্ডের বাইরে ক্লিক করলে
onchange মান পরিবর্তন হলে ইনপুটের মান বদলে ফোকাস হারালে
oninput ইনপুটের মান পরিবর্তন হলে প্রতি অক্ষর টাইপ করার সময়
onsubmit ফর্ম সাবমিট করলে সাবমিট বাটন ক্লিক করলে
onreset ফর্ম রিসেট করলে রিসেট বাটন ক্লিক করলে
onselect টেক্সট সিলেক্ট করলে টেক্সট হাইলাইট করলে

উদাহরণ: ফর্ম ইভেন্ট

<input type="text" 
       onfocus="this.style.border='2px solid blue'"
       onblur="this.style.border='1px solid gray'"
       placeholder="ফোকাস করুন">

<input type="text" 
       oninput="document.getElementById('livePreview').innerText = this.value"
       placeholder="টাইপ করুন">

  <input type="text" placeholder="নাম">
  <button type="submit">সাবমিট</button>
</form>

উইন্ডো/ডকুমেন্ট ইভেন্ট অ্যাট্রিবিউট

উইন্ডো এবং ডকুমেন্ট ইভেন্ট অ্যাট্রিবিউট ব্রাউজার উইন্ডো বা ডকুমেন্টের সাথে সম্পর্কিত।

অ্যাট্রিবিউট বিবরণ কখন ট্রিগার হয়
onload পৃষ্ঠা লোড সম্পন্ন হলে সব কন্টেন্ট লোড হওয়ার পর
onresize উইন্ডোর সাইজ পরিবর্তন হলে ব্রাউজার রিসাইজ করলে
onscroll পৃষ্ঠা স্ক্রোল করলে স্ক্রোল করার সময়
onbeforeunload পৃষ্ঠা ছাড়ার আগে পৃষ্ঠা বন্ধ বা রিফ্রেশ করলে
onerror এরর发生时 ইমেজ লোড না হলে, স্ক্রিপ্ট এরর হলে

উদাহরণ: উইন্ডো ইভেন্ট

<body onload="alert('পৃষ্ঠা লোড সম্পন্ন হয়েছে!')">
  <div onscroll="console.log('স্ক্রোল হচ্ছে')">
    <!-- কন্টেন্ট -->
  </div>
</body>

ড্র্যাগ অ্যান্ড ড্রপ ইভেন্ট অ্যাট্রিবিউট

ড্র্যাগ অ্যান্ড ড্রপ ইভেন্ট অ্যাট্রিবিউট এলিমেন্ট ড্র্যাগ করার সাথে সম্পর্কিত।

অ্যাট্রিবিউট বিবরণ কখন ট্রিগার হয়
ondrag এলিমেন্ট ড্র্যাগ করলে ড্র্যাগ করার সময়
ondragstart ড্র্যাগ শুরু করলে ড্র্যাগ শুরু করার মুহূর্তে
ondragend ড্র্যাগ শেষ করলে ড্র্যাগ শেষ করার সময়
ondragover এলিমেন্টের উপর ড্র্যাগ করলে ড্রপ টার্গেটের উপর ড্র্যাগ করলে
ondrop এলিমেন্টে ড্রপ করলে ড্রপ করার সময়

উদাহরণ: ড্র্যাগ অ্যান্ড ড্রপ

<div draggable="true" 
     ondragstart="event.dataTransfer.setData('text/plain', 'Hello')"
     style="width:100px; height:100px; background:lightblue; display:inline-block; text-align:center; line-height:100px;">
  ড্র্যাগ করুন
</div>

<div ondrop="event.preventDefault(); this.innerText = event.dataTransfer.getData('text/plain')"
     ondragover="event.preventDefault()"
     style="width:150px; height:100px; background:lightgreen; display:inline-block; margin-left:20px; text-align:center; line-height:100px;">
  এখানে ড্রপ করুন
</div>

মিডিয়া ইভেন্ট অ্যাট্রিবিউট

মিডিয়া ইভেন্ট অ্যাট্রিবিউট অডিও, ভিডিও ইত্যাদি মিডিয়া এলিমেন্টের সাথে সম্পর্কিত।

অ্যাট্রিবিউট বিবরণ কখন ট্রিগার হয়
onplay মিডিয়া প্লে শুরু করলে প্লে বাটন চাপলে
onpause মিডিয়া পজ করলে পজ বাটন চাপলে
onended মিডিয়া শেষ হলে ভিডিও/অডিও শেষ হলে
onvolumechange ভলিউম পরিবর্তন হলে ভলিউম বাড়ালে/কমানো হলে

উদাহরণ: মিডিয়া ইভেন্ট

<audio controls 
       onplay="console.log('প্লে শুরু')"
       onpause="console.log('পজ')"
       onended="alert('অডিও শেষ!')">
  <source src="audio.mp3" type="audio/mpeg">
</audio>

ইভেন্ট অ্যাট্রিবিউটের তুলনা

ক্যাটাগরি সবচেয়ে ব্যবহৃত ইভেন্ট ব্যবহারের ক্ষেত্র
মাউস onclick, onmouseenter বাটন ক্লিক, হোভার ইফেক্ট
কীবোর্ড onkeyup, onkeydown সার্চ ফিল্ড, শর্টকাট কী
ফর্ম oninput, onchange লাইভ ভ্যালিডেশন, রিয়েল-টাইম প্রিভিউ
উইন্ডো onload, onscroll পেজ লোড, ইনফিনিটি স্ক্রোল
ড্র্যাগ ondragstart, ondrop ড্র্যাগ অ্যান্ড ড্রপ ফিচার

অনুশীলনী

  1. একটি বাটন তৈরি করুন যা ক্লিক করলে "Hello World!" অ্যালার্ট দেখাবে।
  2. একটি div এলিমেন্ট তৈরি করুন যার উপর মাউস নিয়ে গেলে ব্যাকগ্রাউন্ড কালার নীল হবে এবং মাউস সরিয়ে গেলে আগের রঙে ফিরে যাবে।
  3. একটি ইনপুট ফিল্ড তৈরি করুন যেখানে টাইপ করার সাথে সাথে নিচের একটি প্যারাগ্রাফে টেক্সট লাইভ দেখাবে।
  4. একটি ফর্ম তৈরি করুন যার onSubmit ইভেন্টে অ্যালার্ট দেখাবে এবং ফর্ম সাবমিট বন্ধ করবে।
  5. একটি ইমেজ তৈরি করুন যা লোড না হলে একটি ডিফল্ট ইমেজ দেখাবে (onerror ব্যবহার করে)।
  6. একটি div তৈরি করুন যাতে ড্র্যাগেবল টেক্সট আছে এবং অপর একটি div-এ ড্রপ করলে টেক্সটটি দেখাবে।
  7. নিচের কোডের ভুলগুলো চিহ্নিত করুন এবং সংশোধন করুন: 
    <button onclick=alert('ক্লিক')>ক্লিক</button>

  3. লাইভ প্রিভিউ: 
    <input type="text" oninput="document.getElementById('preview').innerText = this.value">
<p>প্রিভিউ: <span id="preview">-</span></p>
  4. ফর্ম সাবমিট: 
    <form onsubmit="alert('ফর্ম সাবমিট হয়েছে'); return false;">
  <input type="text">
  <button type="submit">সাবমিট</button>
</form>
  5. onerror ইভেন্ট: 
    <img src="invalid.jpg" onerror="this.src='default.jpg'">
  6. ড্র্যাগ অ্যান্ড ড্রপ: 
    <div draggable="true" 
     ondragstart="event.dataTransfer.setData('text/plain', 'ড্র্যাগ করা টেক্সট')"
     style="width:100px; height:100px; background:lightblue;">
  ড্র্যাগ করুন
</div>
<div ondrop="event.preventDefault(); this.innerText = event.dataTransfer.getData('text/plain')"
     ondragover="event.preventDefault()"
     style="width:150px; height:100px; background:lightgreen;">
  ড্রপ করুন
</div>
  7. ভুল সংশোধন: 
    <button onclick="alert('ক্লিক')">ক্লিক</button>
<div onmouseover="this.style.backgroundColor='red'">
<input oninput="console.log('টাইপ')">
<form onsubmit="return false">

    ব্যাখ্যা:
    1. onclick-এর মান কোটেশনের মধ্যে নেই
    2. oninput-এর মানের শেষে কোটেশন নেই
    3. onmouseover সঠিক, onsubmit সঠিক

সংক্ষিপ্ত টিপস

  • কোটেশন ব্যবহার করুন: ইভেন্ট অ্যাট্রিবিউটের মান সবসময় ডাবল বা সিঙ্গেল কোটেশনের মধ্যে দিন।
  • return false ব্যবহার: ফর্ম সাবমিট বা লিংক নেভিগেশন বন্ধ করতে return false ব্যবহার করুন।
  • event অবজেক্ট: ইভেন্ট হ্যান্ডলারে event বা e অবজেক্ট ব্যবহার করে ইভেন্টের বিস্তারিত তথ্য পাওয়া যায়।
  • আধুনিক পদ্ধতি: বড় প্রজেক্টে addEventListener() ব্যবহার করা ভালো, কারণ এটি কোডকে আলাদা রাখে এবং বেশি নিয়ন্ত্রণ দেয়।
  • ডিবাগিং: ইভেন্ট ট্রিগার হচ্ছে কিনা জানতে console.log() ব্যবহার করুন।
  • পারফরম্যান্স: অনেক ইভেন্ট হ্যান্ডলার (যেমন onmousemove, onscroll) পারফরম্যান্সে প্রভাব ফেলতে পারে, সাবধানতার সাথে ব্যবহার করুন।
  • অ্যাক্সেসিবিলিটি: কীবোর্ড ইভেন্টের পাশাপাশি মাউস ইভেন্টও ব্যবহার করুন যাতে সব ব্যবহারকারী সাইট ব্যবহার করতে পারে।
