HTML অ্যাট্রিবিউট (Attributes)

পড়তে সময়: ১৫ মিনিট লেভেল: শিক্ষানবিশ

অ্যাট্রিবিউট কী?

HTML অ্যাট্রিবিউট হল এলিমেন্টের অতিরিক্ত তথ্য যা এলিমেন্টের আচরণ, উপস্থাপনা বা বৈশিষ্ট্য নির্ধারণ করে। অ্যাট্রিবিউট সবসময় শুরু ট্যাগের ভিতরে লেখা হয় এবং এগুলি নাম ও মানের জোড়া আকারে থাকে। অ্যাট্রিবিউট ব্যবহার করে একটি এলিমেন্টকে আরও কাস্টমাইজ করা যায় এবং নির্দিষ্ট কাজ করার নির্দেশ দেওয়া যায়।

অ্যাট্রিবিউট HTML-এর একটি শক্তিশালী বৈশিষ্ট্য যা ওয়েবপৃষ্ঠাকে ডায়নামিক ও ইন্টারঅ্যাকটিভ করতে সাহায্য করে।

অ্যাট্রিবিউটের সিনট্যাক্স

HTML অ্যাট্রিবিউট লেখার একটি নির্দিষ্ট নিয়ম রয়েছে:

মৌলিক সিনট্যাক্স

<এলিমেন্ট নাম অ্যাট্রিবিউট="মান">কন্টেন্ট</এলিমেন্ট>

গঠন:

  • অ্যাট্রিবিউট নাম: কী বৈশিষ্ট্য নির্ধারণ করতে চান (যেমন: href, src, alt)
  • সমান চিহ্ন (=): নাম ও মানের মধ্যে সংযোগ স্থাপন করে
  • মান: অ্যাট্রিবিউটের নির্দিষ্ট মান, সাধারণত উদ্ধৃতি চিহ্নের মধ্যে লেখা হয়

💡 গুরুত্বপূর্ণ: অ্যাট্রিবিউটের মান ডাবল কোটেশন (" ") বা সিঙ্গেল কোটেশন (' ') এর মধ্যে লেখা যায়। ডাবল কোটেশন ব্যবহার করাই স্ট্যান্ডার্ড পদ্ধতি।

উদাহরণ: অ্যাট্রিবিউট ব্যবহার

<a href="https://www.example.com">ভিজিট করুন</a>
<img src="image.jpg" alt="আমার ছবি" width="300" height="200">
<p style="color: blue; font-size: 18px;">নীল রঙের প্যারাগ্রাফ</p>
<input type="text" placeholder="আপনার নাম লিখুন">

সাধারণ HTML অ্যাট্রিবিউট

HTML-এ শতাধিক অ্যাট্রিবিউট রয়েছে। নিচে সবচেয়ে গুরুত্বপূর্ণ এবং বেশি ব্যবহৃত অ্যাট্রিবিউটগুলোর তালিকা দেওয়া হলো:

অ্যাট্রিবিউট বিবরণ উদাহরণ
id এলিমেন্টের ইউনিক আইডি নির্ধারণ করে <div id="header">
class এলিমেন্টের ক্লাস নির্ধারণ করে (CSS বা JS-এর জন্য) <p class="highlight">
style ইনলাইন CSS স্টাইল প্রয়োগ করে <h1 style="color:red">
src ইমেজ, ভিডিও ইত্যাদির সোর্স ফাইল নির্ধারণ করে <img src="photo.jpg">
href লিংকের ইউআরএল নির্ধারণ করে <a href="page.html">
alt ইমেজের বিকল্প টেক্সট (অ্যাক্সেসিবিলিটির জন্য) <img alt="বর্ণনা">
title টুলটিপ হিসেবে দেখানো অতিরিক্ত তথ্য <abbr title="HyperText Markup Language">HTML</abbr>
width / height এলিমেন্টের প্রস্থ ও উচ্চতা নির্ধারণ করে <img width="100" height="100">
type ইনপুট ফিল্ডের ধরন নির্ধারণ করে <input type="email">
placeholder ইনপুট ফিল্ডে প্লেসহোল্ডার টেক্সট দেখায় <input placeholder="নাম লিখুন">
required ইনপুট ফিল্ড পূরণ করা আবশ্যক নির্দেশ করে <input required>
disabled এলিমেন্ট নিষ্ক্রিয় করে দেয় <button disabled>ক্লিক করুন</button>
target লিংক কোথায় খুলবে তা নির্ধারণ করে <a target="_blank">

বিভিন্ন ধরনের অ্যাট্রিবিউট

অ্যাট্রিবিউট তিনটি প্রধান ক্যাটাগরিতে বিভক্ত:

১. বাধ্যতামূলক অ্যাট্রিবিউট

কিছু অ্যাট্রিবিউট নির্দিষ্ট এলিমেন্টের জন্য বাধ্যতামূলক। যেমন <img> ট্যাগে src এবং alt অ্যাট্রিবিউট থাকা আবশ্যক।

<!-- সঠিক -->
<img src="logo.png" alt="কোম্পানির লোগো">

<!-- ভুল (alt নেই) -->
<img src="logo.png">

২. বৈশ্বিক অ্যাট্রিবিউট (Global Attributes)

বৈশ্বিক অ্যাট্রিবিউট সব HTML এলিমেন্টের সাথে ব্যবহার করা যায়। যেমন: id, class, style, title, lang, hidden ইত্যাদি।

<p id="intro" class="text" style="color:green" title="এটি একটি প্যারাগ্রাফ">
  বৈশ্বিক অ্যাট্রিবিউট ব্যবহার করা হয়েছে।
</p>
<p hidden>এই প্যারাগ্রাফটি লুকানো আছে।</p>

বৈশ্বিক অ্যাট্রিবিউট ব্যবহার করা হয়েছে।

এই প্যারাগ্রাফটি লুকানো আছে।

৩. ইভেন্ট হ্যান্ডলার অ্যাট্রিবিউট

জাভাস্ক্রিপ্ট ব্যবহার করে বিভিন্ন ইভেন্ট হ্যান্ডল করার জন্য এই অ্যাট্রিবিউটগুলো ব্যবহার করা হয়।

<button onclick="alert('হ্যালো!')">ক্লিক করুন</button>
<input onfocus="this.style.backgroundColor='yellow'">

বুলিয়ান অ্যাট্রিবিউট

কিছু অ্যাট্রিবিউটের মান থাকে না। এদের বুলিয়ান অ্যাট্রিবিউট বলা হয়। এগুলি উপস্থিত থাকলেই সেটি সত্য (true) হিসেবে বিবেচিত হয়।

উদাহরণ: বুলিয়ান অ্যাট্রিবিউট

<!-- চেকবক্স প্রি-সিলেক্টেড -->
<input type="checkbox" checked> আমি সম্মতি দিচ্ছি

<!-- ইনপুট ফিল্ড রিডঅনলি -->
<input type="text" value="পড়া যায় শুধু" readonly>

<!-- বাটন ডিসএবলড -->
<button disabled>সাবমিট করুন</button>

প্রিভিউ:

আমি সম্মতি দিচ্ছি



কাস্টম অ্যাট্রিবিউট (data-*)

HTML5-এ কাস্টম অ্যাট্রিবিউট তৈরি করার সুবিধা রয়েছে। এগুলি data- দিয়ে শুরু হয় এবং জাভাস্ক্রিপ্টে ডেটা সংরক্ষণের জন্য ব্যবহার করা হয়।

উদাহরণ: data-* অ্যাট্রিবিউট

<div data-user-id="12345" data-role="admin" data-theme="dark">
  ব্যবহারকারীর তথ্য
</div>

<button data-product-id="100" data-price="500" onclick="showProduct(this)">
  পণ্য দেখুন
</button>

<script>
function showProduct(button) {
  let productId = button.dataset.productId;
  let price = button.dataset.price;
  alert('পণ্য আইডি: ' + productId + ', দাম: ' + price + ' টাকা');
}
</script>
ব্যবহারকারীর তথ্য

💡 কাস্টম অ্যাট্রিবিউট জাভাস্ক্রিপ্টে ডেটা সংরক্ষণ ও ব্যবহারের জন্য খুবই কার্যকরী।

অ্যাট্রিবিউটের গুরুত্বপূর্ণ টিপস

  • অ্যাট্রিবিউট মান সবসময় উদ্ধৃতি চিহ্নে দিন: HTML5-এ উদ্ধৃতি ছাড়া লেখা যায়, তবে ভালো অভ্যাস হল ডাবল কোটেশন ব্যবহার করা।
  • অ্যাট্রিবিউট নাম ছোট হাতের অক্ষরে লিখুন: HTML5-এ কেস সেন্সিটিভ নয়, কিন্তু ছোট হাতের অক্ষর ব্যবহার করা স্ট্যান্ডার্ড।
  • একাধিক অ্যাট্রিবিউট স্পেস দিয়ে আলাদা করুন: ক্রম যেকোনো হতে পারে, তবে পঠনযোগ্যতার জন্য যুক্তিসঙ্গত ক্রম অনুসরণ করুন।
  • বুলিয়ান অ্যাট্রিবিউটে মান না দিয়ে শুধু নাম দিন: যেমন checked, disabled।
  • কাস্টম অ্যাট্রিবিউটের জন্য data-* ব্যবহার করুন: এটি HTML ভ্যালিডেটর দ্বারা স্বীকৃত এবং ভবিষ্যতে কোন সমস্যা তৈরি করবে না।

অনুশীলনী

  1. একটি লিংক তৈরি করুন যা নতুন ট্যাবে খুলবে। সঠিক অ্যাট্রিবিউট ব্যবহার করুন।
  2. একটি ইমেজ যোগ করুন যার প্রস্থ 200px, উচ্চতা 150px, এবং alt টেক্সট আছে।
  3. একটি ফর্ম তৈরি করুন যাতে একটি নামের ইনপুট ফিল্ড আছে যা পূরণ করা আবশ্যক (required)।
  4. একটি প্যারাগ্রাফ তৈরি করুন যার id="about", class="content", এবং style="background-color:lightgray" আছে।
  5. একটি বাটন তৈরি করুন যা ক্লিক করলে alert দেখাবে (onclick অ্যাট্রিবিউট ব্যবহার করে)।
  6. একটি div এলিমেন্ট তৈরি করুন যাতে data-role="admin" এবং data-status="active" কাস্টম অ্যাট্রিবিউট আছে।
  7. নিচের কোডে অ্যাট্রিবিউটের ভুলগুলো চিহ্নিত করুন এবং সংশোধন করুন: 
    <img src=photo.jpg alt="ছবি width="300">
<a href='page.html target=_blank>লিংক</a>
<input type=text placeholder="নাম লিখুন required>
সমাধান দেখুন
  1. নতুন ট্যাবে খোলার লিংক: 
    <a href="https://www.example.com" target="_blank">ভিজিট করুন</a>
  2. ইমেজ অ্যাট্রিবিউট: 
    <img src="image.jpg" alt="ছবির বিবরণ" width="200" height="150">
  3. required সহ ফর্ম: 
    <form>
  <input type="text" placeholder="আপনার নাম" required>
  <button type="submit">সাবমিট</button>
</form>
  4. প্যারাগ্রাফ অ্যাট্রিবিউট: 
    <p id="about" class="content" style="background-color:lightgray">
  এটি একটি প্যারাগ্রাফ।
</p>
  5. onclick সহ বাটন: 
    <button onclick="alert('হ্যালো!')">ক্লিক করুন</button>
  6. data-* অ্যাট্রিবিউট: 
    <div data-role="admin" data-status="active">
  অ্যাডমিন প্যানেল
</div>
  7. ভুল সংশোধন: 
    <img src="photo.jpg" alt="ছবি" width="300">
<a href="page.html" target="_blank">লিংক</a>
<input type="text" placeholder="নাম লিখুন" required>

    ব্যাখ্যা: প্রথমটিতে alt-এর কোটেশন সঠিকভাবে বন্ধ হয়নি, দ্বিতীয়টিতে href-এর কোটেশন ও target-এর মান সঠিক নয়, তৃতীয়টিতে type-এর মান কোটেশনে নেই এবং required-এর মান দেওয়া ভুল (বুলিয়ান অ্যাট্রিবিউটের মান লাগে না)।

সংক্ষিপ্ত টিপস

  • অ্যাক্সেসিবিলিটি নিশ্চিত করুন: সব ইমেজে alt অ্যাট্রিবিউট ব্যবহার করুন। এটি স্ক্রিন রিডার ব্যবহারকারীদের জন্য গুরুত্বপূর্ণ।
  • লিংকের জন্য title ব্যবহার করুন: লিংকের গন্তব্য সম্পর্কে অতিরিক্ত তথ্য দিতে title অ্যাট্রিবিউট ব্যবহার করুন।
  • id ইউনিক রাখুন: একটি পৃষ্ঠায় একই id একাধিকবার ব্যবহার করবেন না।
  • class পুনরায় ব্যবহার করুন: একই স্টাইল একাধিক এলিমেন্টে দিতে class ব্যবহার করুন।
  • ইনলাইন স্টাইল এড়িয়ে চলুন: CSS আলাদা ফাইলে রাখা ভালো, তবে দ্রুত পরীক্ষার জন্য style অ্যাট্রিবিউট ব্যবহার করা যায়।
  • সঠিক ডকটাইপ ব্যবহার করুন: HTML5-এ অ্যাট্রিবিউট লেখার নিয়ম কিছুটা নমনীয়, তবে সঠিক ডকটাইপ থাকা জরুরি।
