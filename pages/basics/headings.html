হেডিং ট্যাগ (h1 - h6)

পড়তে সময়: ১০ মিনিট লেভেল: শিক্ষানবিশ

হেডিং ট্যাগ কী?

হেডিং ট্যাগ বা শিরোনাম ট্যাগ HTML-এর সবচেয়ে গুরুত্বপূর্ণ এলিমেন্টগুলোর একটি। এগুলো ওয়েবপৃষ্ঠার শিরোনাম ও সাব-শিরোনাম নির্ধারণ করে। হেডিং ট্যাগ ৬টি: h1, h2, h3, h4, h5, h6। h1 সবচেয়ে গুরুত্বপূর্ণ ও বড়, h6 সবচেয়ে কম গুরুত্বপূর্ণ ও ছোট।

ব্রাউজার সাধারণত h1-কে সবচেয়ে বড় আকারে এবং h6-কে সবচেয়ে ছোট আকারে দেখায়। তবে CSS ব্যবহার করে এদের সাইজ পরিবর্তন করা যায়।

হেডিং ট্যাগের ব্যবহার

HTML-এ হেডিং ট্যাগ ব্যবহার করা খুবই সহজ। নিচের উদাহরণে বিভিন্ন লেভেলের হেডিং দেখানো হয়েছে:

উদাহরণ: বিভিন্ন লেভেলের হেডিং

<h1>এটি প্রধান শিরোনাম (h1)</h1>
<h2>এটি সাব-শিরোনাম (h2)</h2>
<h3>এটি ছোট শিরোনাম (h3)</h3>
<h4>এটি আরও ছোট শিরোনাম (h4)</h4>
<h5>এটি খুব ছোট শিরোনাম (h5)</h5>
<h6>এটি সবচেয়ে ছোট শিরোনাম (h6)</h6>

এটি সবচেয়ে ছোট শিরোনাম (h6)

হেডিং হায়ারারকি (শ্রেণিবিন্যাস)

হেডিং হায়ারারকি বজায় রাখা SEO এবং অ্যাক্সেসিবিলিটির জন্য খুবই গুরুত্বপূর্ণ। সঠিকভাবে স্ট্রাকচার্ড হেডিং সার্চ ইঞ্জিনকে আপনার কন্টেন্ট বুঝতে সাহায্য করে এবং স্ক্রিন রিডার ব্যবহারকারীদের জন্য নেভিগেশন সহজ করে।

💡 গুরুত্বপূর্ণ নিয়ম: প্রতিটি পৃষ্ঠায় শুধুমাত্র একটি <h1> থাকা উচিত। h1 পৃষ্ঠার মূল বিষয় নির্দেশ করে।

  • ক্রম অনুসরণ করুন: h1 → h2 → h3 → h4 → h5 → h6 ক্রমান্বয়ে ব্যবহার করতে হবে।
  • লেভেল স্কিপ করবেন না: h1-এর পর সরাসরি h3 ব্যবহার করা উচিত নয়।
  • একাধিক h1 এড়িয়ে চলুন: একটি পৃষ্ঠায় একাধিক h1 ব্যবহার করলে SEO-তে নেতিবাচক প্রভাব পড়তে পারে।

SEO-তে হেডিং-এর গুরুত্ব

সার্চ ইঞ্জিন (যেমন গুগল) হেডিং ট্যাগ ব্যবহার করে আপনার কন্টেন্টের গঠন বুঝতে চেষ্টা করে। সঠিকভাবে হেডিং ব্যবহার করলে আপনার ওয়েবপৃষ্ঠা সার্চ রেজাল্টে ভালো অবস্থান করতে পারে।

💡 SEO টিপস:

  • h1-এ মূল কীওয়ার্ড ব্যবহার করুন: h1-এ আপনার পৃষ্ঠার প্রধান বিষয়বস্তুর সাথে সম্পর্কিত কীওয়ার্ড রাখুন।
  • হেডিং অর্থপূর্ণ হোক: হেডিং বাক্যের মতো বর্ণনামূলক হওয়া উচিত। যেমন "আমার ব্লগ" না লিখে "২০২৬ সালের সেরা ওয়েব ডেভেলপমেন্ট টিপস" লেখা ভালো।
  • কীওয়ার্ড স্টাফিং এড়িয়ে চলুন: অস্বাভাবিকভাবে বারবার একই কীওয়ার্ড ব্যবহার করবেন না।
  • প্রতিটি h2 আলাদা বিষয় কভার করুক: প্রতিটি সাব-সেকশন একটি নির্দিষ্ট বিষয় নিয়ে আলোচনা করবে।

অ্যাক্সেসিবিলিটি এবং স্ক্রিন রিডার

স্ক্রিন রিডার ব্যবহারকারীরা (যারা দৃষ্টি প্রতিবন্ধী) হেডিং ট্যাগ ব্যবহার করে পৃষ্ঠায় দ্রুত নেভিগেট করে। তারা সাধারণত হেডিংগুলোর তালিকা তৈরি করে এবং সেখান থেকে নির্দিষ্ট সেকশনে যেতে পারে।

সঠিক হেডিং স্ট্রাকচার থাকলে স্ক্রিন রিডার ব্যবহারকারীরা পৃষ্ঠার বিষয়বস্তু সহজেই বুঝতে পারে এবং প্রয়োজনীয় তথ্যে দ্রুত পৌঁছাতে পারে।

হেডিং-এ স্টাইলিং (CSS)

হেডিং ট্যাগের ডিফল্ট সাইজ CSS ব্যবহার করে পরিবর্তন করা যায়। নিচের উদাহরণ দেখুন:

উদাহরণ: কাস্টম হেডিং স্টাইল

<h1 style="color: #2c3e50; font-size: 36px; border-bottom: 3px solid #3498db;">
  কাস্টম স্টাইলের শিরোনাম
</h1>
<h2 style="color: #16a085; background-color: #f0f0f0; padding: 10px;">
  ব্যাকগ্রাউন্ড সহ হেডিং
</h2>

হেডিং ট্যাগের তুলনা

ট্যাগ গুরুত্ব ডিফল্ট সাইজ ব্যবহার
<h1> সর্বোচ্চ 2em (32px) পৃষ্ঠার প্রধান শিরোনাম
<h2> উচ্চ 1.5em (24px) প্রধান সেকশনের শিরোনাম
<h3> মাঝারি 1.17em (18.72px) সাব-সেকশনের শিরোনাম
<h4> নিম্ন-মাঝারি 1em (16px) ছোট সাব-সেকশন
<h5> নিম্ন 0.83em (13.28px) বিরল ব্যবহার
<h6> সর্বনিম্ন 0.67em (10.72px) সবচেয়ে কম গুরুত্বপূর্ণ

অনুশীলনী

  1. একটি HTML ডকুমেন্ট তৈরি করুন যেখানে আপনার নাম ও ঠিকানা h1, h2, h3 ট্যাগ ব্যবহার করে দেখানো হয়েছে।
  2. একটি ব্লগ পোস্টের স্ট্রাকচার তৈরি করুন (শিরোনাম, সাব-শিরোনাম, অনুচ্ছেদ) সঠিক হেডিং হায়ারারকি ব্যবহার করে।
  3. নিচের ভুল স্ট্রাকচারটি সংশোধন করুন: h1 → h3 → h2 → h4
  4. একটি পৃষ্ঠায় একাধিক h1 ব্যবহার করলে কী সমস্যা হতে পারে? উদাহরণসহ ব্যাখ্যা করুন।
  5. CSS ব্যবহার করে একটি h1 এলিমেন্টের জন্য নিচের স্টাইল প্রয়োগ করুন: নীল রঙ, 40px ফন্ট সাইজ, নিচে লাল রঙের বর্ডার।
সমাধান দেখুন
  1. উদাহরণ: 
    <h1>জন Doe</h1>
<h2>ঠিকানা</h2>
<h3>ঢাকা, বাংলাদেশ</h3>
  2. ব্লগ পোস্ট স্ট্রাকচার: 
    <h1>HTML শেখার ১০টি সহজ টিপস</h1>
  <h2>১. বেসিক দিয়ে শুরু করুন</h2>
    <h3>HTML স্ট্রাকচার বোঝা</h3>
  <h2>২. নিয়মিত প্র্যাকটিস করুন</h2>
    <h3>ছোট ছোট প্রজেক্ট তৈরি করুন</h3>
  3. সংশোধিত স্ট্রাকচার: 
    h1 → h2 → h3 → h4
  4. একাধিক h1 ব্যবহারের সমস্যা: সার্চ ইঞ্জিন বুঝতে পারে না কোনটি মূল শিরোনাম, ফলে SEO-তে নেতিবাচক প্রভাব পড়ে। স্ক্রিন রিডার ব্যবহারকারীরাও বিভ্রান্ত হয়।
  5. CSS স্টাইল: 
    <h1 style="color: blue; font-size: 40px; border-bottom: 3px solid red;">শিরোনাম</h1>

সংক্ষিপ্ত টিপস

  • প্রতি পৃষ্ঠায় একটি h1 ব্যবহার করুন: এটি পৃষ্ঠার মূল বিষয় নির্দেশ করে।
  • হেডিং স্ট্রাকচার আউটলাইন তৈরি করুন: কন্টেন্ট লেখার আগে হেডিংগুলোর একটি আউটলাইন তৈরি করে নিন।
  • সংক্ষিপ্ত ও স্পষ্ট হেডিং ব্যবহার করুন: হেডিং যেন খুব লম্বা না হয় এবং বিষয়বস্তু পরিষ্কারভাবে বুঝায়।
  • মোবাইল ডিভাইসে পরীক্ষা করুন: হেডিং মোবাইল স্ক্রিনেও ঠিকমতো দেখায় কিনা চেক করুন।
  • CSS দিয়ে ডিজাইন উন্নত করুন: ডিফল্ট হেডিং স্টাইলের চেয়ে CSS ব্যবহার করে আরও আকর্ষণীয় ডিজাইন তৈরি করুন।
