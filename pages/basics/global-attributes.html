গ্লোবাল অ্যাট্রিবিউট (Global Attributes)

পড়তে সময়: ১৮ মিনিট লেভেল: মাঝারি

গ্লোবাল অ্যাট্রিবিউট কী?

গ্লোবাল অ্যাট্রিবিউট হল সেই সকল অ্যাট্রিবিউট যা HTML-এর যেকোনো এলিমেন্টের সাথে ব্যবহার করা যায়। এগুলি সব এলিমেন্টের জন্য সাধারণ বৈশিষ্ট্য নির্ধারণ করে। HTML5-এ প্রচুর গ্লোবাল অ্যাট্রিবিউট সংযোজন করা হয়েছে যা ডেভেলপারদের আরও বেশি কন্ট্রোল দেয়।

গ্লোবাল অ্যাট্রিবিউট ব্যবহার করে এলিমেন্টের আইডি, ক্লাস, স্টাইল, ভাষা, ড্র্যাগেবিলিটি, স্পেলচেক, কাস্টম ডেটা ইত্যাদি নিয়ন্ত্রণ করা যায়।

💡 গুরুত্বপূর্ণ: গ্লোবাল অ্যাট্রিবিউট সব HTML এলিমেন্টের সাথে ব্যবহার করা যায়, তবে কিছু এলিমেন্টে কিছু অ্যাট্রিবিউটের কোনো প্রভাব নাও থাকতে পারে।

মৌলিক গ্লোবাল অ্যাট্রিবিউট

নিচে সবচেয়ে বেশি ব্যবহৃত গ্লোবাল অ্যাট্রিবিউটগুলোর তালিকা ও ব্যবহার দেখানো হলো:

অ্যাট্রিবিউট বিবরণ উদাহরণ
id এলিমেন্টের ইউনিক আইডেন্টিফায়ার <div id="header">
class এলিমেন্টের ক্লাস নাম (একাধিক হতে পারে) <p class="highlight bold">
style ইনলাইন CSS স্টাইল <h1 style="color:red">
title টুলটিপ হিসেবে অতিরিক্ত তথ্য <abbr title="World Wide Web">WWW</abbr>
lang এলিমেন্টের ভাষা নির্ধারণ <p lang="bn">বাংলা টেক্সট</p>
dir টেক্সটের দিকনির্দেশ (ltr/rtl) <p dir="rtl">আরবি টেক্সট</p>
hidden এলিমেন্ট লুকিয়ে রাখে <div hidden>লুকানো</div>
tabindex কীবোর্ড নেভিগেশনের ক্রম <input tabindex="1">
accesskey শর্টকাট কী নির্ধারণ <a accesskey="h">হোম</a>
contenteditable এলিমেন্ট এডিট করা যাবে কিনা <div contenteditable="true">এডিট করুন</div>

id এবং class অ্যাট্রিবিউট

id এবং class হল সবচেয়ে বেশি ব্যবহৃত গ্লোবাল অ্যাট্রিবিউট। এগুলি CSS এবং JavaScript-এর সাথে এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয়।

উদাহরণ: id এবং class ব্যবহার

<!-- id ইউনিক হতে হবে -->
<div id="main-content">
  <h1 id="page-title">স্বাগতম</h1>
  <p class="intro">এটি একটি পরিচিতি প্যারাগ্রাফ।</p>
  <p class="intro highlight">এটি আরেকটি প্যারাগ্রাফ।</p>
</div>

<style>
  #page-title { color: navy; }
  .intro { font-size: 18px; }
  .highlight { background-color: yellow; }
</style>

প্রিভিউ:

স্বাগতম

এটি একটি পরিচিতি প্যারাগ্রাফ।

এটি আরেকটি প্যারাগ্রাফ।

💡 টিপস: id শুধুমাত্র একবার ব্যবহার করা যায়, কিন্তু class একাধিক এলিমেন্টে এবং একটি এলিমেন্টে একাধিক class ব্যবহার করা যায় (স্পেস দিয়ে আলাদা)।

title অ্যাট্রিবিউট

title অ্যাট্রিবিউট এলিমেন্টের উপর মাউস নিয়ে গেলে টুলটিপ হিসেবে অতিরিক্ত তথ্য দেখায়। এটি অ্যাক্সেসিবিলিটি এবং ব্যবহারকারীর অভিজ্ঞতা উন্নত করে।

hidden অ্যাট্রিবিউট এলিমেন্টটিকে ব্রাউজারে লুকিয়ে রাখে। এটি বুলিয়ান অ্যাট্রিবিউট, অর্থাৎ উপস্থিত থাকলেই এলিমেন্ট লুকানো হয়।

tabindex অ্যাট্রিবিউট

tabindex অ্যাট্রিবিউট কীবোর্ড ট্যাব নেভিগেশনের ক্রম নির্ধারণ করে। এটি অ্যাক্সেসিবিলিটির জন্য গুরুত্বপূর্ণ।

উদাহরণ: tabindex ব্যবহার

<!-- ট্যাব ক্রম: 1, 2, 3 -->
<input type="text" tabindex="1" placeholder="প্রথমে এখানে ফোকাস">
<input type="text" tabindex="3" placeholder="তৃতীয়">
<button tabindex="2">দ্বিতীয়</button>

<!-- tabindex="-1" ট্যাব দিয়ে ফোকাস করা যায় না -->
<div tabindex="-1">ট্যাব দিয়ে ফোকাস করা যাবে না</div>

<!-- tabindex="0" স্বাভাবিক ক্রমে থাকে -->
<a href="#" tabindex="0">স্বাভাবিক লিংক</a>

💡 টিপস: tabindex="0" এলিমেন্টকে স্বাভাবিক ট্যাব অর্ডারে রাখে। tabindex="-1" এলিমেন্টকে ট্যাব অর্ডার থেকে বাদ দেয় কিন্তু JavaScript দিয়ে ফোকাস করা যায়।

contenteditable অ্যাট্রিবিউট

contenteditable অ্যাট্রিবিউট ব্যবহার করে এলিমেন্টের কন্টেন্টকে এডিটেবল করা যায়। এটি রিচ টেক্সট এডিটর বা ইনলাইন এডিটিং ফিচার তৈরি করতে ব্যবহৃত হয়।

উদাহরণ: contenteditable ব্যবহার

<div contenteditable="true" 
     style="border:2px solid #ccc; padding:10px; min-height:100px;">
  এই টেক্সটটি আপনি এডিট করতে পারেন। ক্লিক করে লিখুন!
</div>

<p contenteditable="true">এই প্যারাগ্রাফটিও এডিট করা যায়।</p>

accesskey অ্যাট্রিবিউট

accesskey অ্যাট্রিবিউট ব্যবহার করে এলিমেন্টের জন্য শর্টকাট কী নির্ধারণ করা যায়।

উদাহরণ: accesskey ব্যবহার

<a href="index.html" accesskey="h">হোম (Alt+H)</a>
<a href="about.html" accesskey="a">আমাদের সম্পর্কে (Alt+A)</a>
<a href="contact.html" accesskey="c">যোগাযোগ (Alt+C)</a>

     Windows: Alt + accesskey
     Mac: Ctrl + Option + accesskey -->
হোম (Alt+H) আমাদের সম্পর্কে (Alt+A) যোগাযোগ (Alt+C)

spellcheck অ্যাট্রিবিউট

spellcheck অ্যাট্রিবিউট ব্রাউজারের স্পেল চেকিং ফিচার চালু বা বন্ধ করে।

<!-- স্পেল চেক অন -->
<textarea spellcheck="true" rows="3" cols="40">
This is a sample text with speling error.
</textarea>

<!-- স্পেল চেক অফ -->
<input type="text" spellcheck="false" value="This is speling error">

data-* কাস্টম অ্যাট্রিবিউট

data-* অ্যাট্রিবিউট ব্যবহার করে কাস্টম ডেটা সংরক্ষণ করা যায়। এটি জাভাস্ক্রিপ্টে ডেটা পাস করার জন্য খুবই কার্যকরী।

উদাহরণ: data-* অ্যাট্রিবিউট

<div data-user-id="12345" data-role="admin" data-theme="dark">
  ব্যবহারকারীর তথ্য
</div>

<button data-product-id="100" data-price="500" onclick="showInfo(this)">
  পণ্যের তথ্য
</button>

<script>
function showInfo(element) {
  let productId = element.dataset.productId;
  let price = element.dataset.price;
  alert(`পণ্য ID: ${productId}, দাম: ${price} টাকা`);
}
</script>
ব্যবহারকারীর তথ্য (ডেটা স্টোর করা আছে)

গ্লোবাল অ্যাট্রিবিউটের তুলনা

অ্যাট্রিবিউট ব্যবহারের ক্ষেত্র গুরুত্বপূর্ণ বৈশিষ্ট্য
id CSS/JS সিলেক্টর, URL fragment পৃষ্ঠায় ইউনিক হতে হবে
class CSS স্টাইলিং, JS সিলেক্টর একাধিক মান স্পেস দিয়ে আলাদা
title টুলটিপ, অতিরিক্ত তথ্য অ্যাক্সেসিবিলিটি উন্নত করে
hidden এলিমেন্ট লুকানো বুলিয়ান অ্যাট্রিবিউট
contenteditable ইনলাইন এডিটিং true/false মান নেয়
tabindex কীবোর্ড নেভিগেশন অ্যাক্সেসিবিলিটির জন্য গুরুত্বপূর্ণ
data-* কাস্টম ডেটা স্টোরেজ জাভাস্ক্রিপ্টে সহজে অ্যাক্সেস

অনুশীলনী

  1. একটি div এলিমেন্ট তৈরি করুন যার id="container", class="main-box highlight", এবং title="এটি মূল কন্টেইনার" আছে।
  2. একটি প্যারাগ্রাফ তৈরি করুন যা বাংলা ভাষায় লেখা এবং ডান থেকে বাম দিকে সাজানো (rtl)।
  3. একটি বাটন তৈরি করুন যা ক্লিক করলে একটি hidden প্যারাগ্রাফ দৃশ্যমান হবে।
  4. একটি div এলিমেন্ট তৈরি করুন যা contenteditable এবং এতে একটি নোট লেখার জায়গা আছে।
  5. তিনটি লিংক তৈরি করুন যাদের accesskey যথাক্রমে "h", "a", "c" আছে।
  6. একটি এলিমেন্ট তৈরি করুন যাতে data-user="rahim" এবং data-role="editor" কাস্টম অ্যাট্রিবিউট আছে।
  7. নিচের কোডের ভুলগুলো চিহ্নিত করুন এবং সংশোধন করুন: 
    <div id="box" id="main">
<p class="text text-bold text-blue">
<div hidden="true">
<input tabindex="-1">
<div contenteditable>এডিট করুন</div>
সমাধান দেখুন
  1. div with id, class, title: 
    <div id="container" class="main-box highlight" title="এটি মূল কন্টেইনার">
  কন্টেন্ট
</div>
  2. বাংলা ও RTL প্যারাগ্রাফ: 
    <p lang="bn" dir="rtl">
এটি একটি বাংলা প্যারাগ্রাফ যা ডান থেকে বাম দিকে সাজানো।
</p>
  3. hidden দেখানোর বাটন: 
    <button onclick="document.getElementById('secret').hidden = false">
  বার্তা দেখান
</button>
<p id="secret" hidden>এটি একটি গোপন বার্তা!</p>

    <div contenteditable="true" style="border:1px solid #ccc; padding:10px; min-height:100px;">
আপনার নোট এখানে লিখুন...
</div>
  5. accesskey সহ লিংক: 
    <a href="#" accesskey="h">হোম (Alt+H)</a>
<a href="#" accesskey="a">আমাদের সম্পর্কে (Alt+A)</a>
<a href="#" accesskey="c">যোগাযোগ (Alt+C)</a>
  6. data-* অ্যাট্রিবিউট: 
    <div data-user="rahim" data-role="editor">
  রহিমের তথ্য
</div>
  7. ভুল সংশোধন: 
    <div id="box">
<p class="text text-bold text-blue">
<div hidden>
<input tabindex="-1">
<div contenteditable="true">এডিট করুন</div>

    ব্যাখ্যা:
    1. একই এলিমেন্টে একাধিক id ব্যবহার করা যাবে না
    2. hidden বুলিয়ান অ্যাট্রিবিউটের মান লাগে না
    3. contenteditable-এর মান স্পষ্টভাবে true/false দেওয়া ভালো

সংক্ষিপ্ত টিপস

  • id ইউনিক রাখুন: একটি পৃষ্ঠায় একই id একাধিকবার ব্যবহার করবেন না।
  • class পুনরায় ব্যবহার করুন: একাধিক এলিমেন্টে একই class ব্যবহার করতে পারেন।
  • title অ্যাক্সেসিবিলিটি: স্ক্রিন রিডার title অ্যাট্রিবিউট পড়ে, তাই অর্থপূর্ণ তথ্য দিন।
  • lang অ্যাট্রিবিউট ব্যবহার করুন: এটি SEO এবং অ্যাক্সেসিবিলিটির জন্য গুরুত্বপূর্ণ।
  • hidden বনাম CSS display:none: hidden সিমান্টিকভাবে লুকানোর জন্য, CSS বেশি কন্ট্রোল দেয়।
  • tabindex ব্যবহার সাবধানে করুন: অতিরিক্ত tabindex ব্যবহার ব্যবহারকারীর অভিজ্ঞতা খারাপ করতে পারে।
  • data-* নেমিং: data- অ্যাট্রিবিউটের নাম ছোট হাতের অক্ষরে এবং হাইফেন দিয়ে লেখা উচিত।
  • contenteditable সিকিউরিটি: ব্যবহারকারীর ইনপুট স্যানিটাইজ করতে ভুলবেন না।
