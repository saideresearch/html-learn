HTML5 Fundamentals

HTML is the standard language used to create web pages. HTML5 is the latest version, providing modern features and better support for multimedia. Every HTML document follows a basics structure: doctype, html, head, and body. Web pages are built in using elements (tags), which define content. Attributes provide additional informatiion to elements. Elements can be nested inside each other to create structure. Comments are used to add notes in the code that are not displayed in the browser.

Example:

				
					<!DOCTYPE html>
					<html lang="en">
					<head>
					<meta charset="UTF-8">
					<title>My First Page</title>
					</head>
					<body>
					<!-- This is a heading -->
					<h1 class="title">Hello World</h1>
					<!-- This is a paragraph -->
					<p>This is my first HTML5 page.</p>
					</body>
					</html>

Document Metadata

The head element contains metadata about the document. The title defines the page title shown in the browser tab. Meta tags provide information like character encoding, responsiveness, and SEO details. The base tag sets a base URL for all relative links. The link tag is used to connect external resources like stylesheets and favicon. The style tag is used to write internal CSS. The script tag is used to add JavaScript to the pages.

Example:

					
					<!DOCTYPE html>
					<html lang="en">
					<head>
					<!-- Page title -->
					<title>Document Metadata Example</title>
					<!-- Meta tags -->
					<meta charset="UTF-8">
					<meta name="viewport" content="width=device-width, initial-scan=1.0">
					<meta name="description" content="Example of HTML5 metadata">
					<meta name="keywords" content=HTML5, metadata, tutorial">
					<meta name="author" content="Saide Hossain">
					<!-- Base URL -->
					<base href="https://example.com/">
					<!-- External resources -->
					<link rel="icon" href="favicon.ico">
					<link rel="stylesheet" href="styles.css">
					<!-- Internal CSS -->
					<style>
						body {
							font-family: Arial, sans-serif;
						}
					</style>
					<script>
						console.log("Page loaded");
					</script>
					</head>
					<body>
						<h1>Metadata Example Page</h1>
					</body>
					</html>
