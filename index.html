HTML5 ফান্ডামেন্টালস

HTML হলো ওয়েবপেজ তৈরী করার স্ট্যান্ডার্ড ল্যাংগুয়েজ। HTML5 is the latest version, যা modern features এবং multimedia এর জন্য better support দেয়। প্রতিটি HTML document একটি basic structure follow করে: doctype, html, head, এবং body। Web pages তৈরি হয় elements (tags) ব্যবহার করে, যা content define করে। Attributes elements-এর অতিরিক্ত information প্রদান করে। Elements একটির ভিতরে আরেকটি nested হয়ে structure তৈরি করতে পারে। Comments কোডের ভিতরে note যোগ করতে ব্যবহার করা হয়, যা browser-এ প্রদর্শিত হয় না।

Example:

				
					<!DOCTYPE html>
					<html lang="en">
					<head>
					<meta charset="UTF-8">
					<title>My First Page</title>
					</head>
					<body>
					<!-- This is a heading -->
					<h1 class="title">Hello World</h1>
					<!-- This is a paragraph -->
					<p>This is my first HTML5 page.</p>
					</body>
					</html>

ডকুমেন্ট মেটাডাটা

head element ডকুমেন্টের মেটাডাটা ধারণ করে, যা পেজ সম্পর্কে অতিরিক্ত তথ্য ব্রাউজারকে জানায়। title ব্রাউজার ট্যাব-এ প্রদর্শিত পেজের নাম নির্ধারণ করে। meta tags ক্যারেক্টার এনকোডিং, রেসপনসিভনেস এবং এসইও সম্পর্কিত গুরুত্বপূর্ণ তথ্য প্রদান করে। base tag সব রিলেটিভ লিংকের জন্য একটি বেস ইউআরএল নির্ধারণ করে। link tag এক্সটার্নাল রিসোর্স যেমন স্টাইলশিট এবং ফ্যাভিকন যুক্ত করতে ব্যবহার করা হয়। style tag ইন্টারনাল সিএসএস লেখার জন্য ব্যবহার করা হয়। script tag পেজে জাভাস্ক্রিপ্ট যোগ করার জন্য ব্যবহার করা হয়।

Example:

					
					<!DOCTYPE html>
					<html lang="en">
					<head>
					<!-- Page title -->
					<title>Document Metadata Example</title>
					<!-- Meta tags -->
					<meta charset="UTF-8">
					<meta name="viewport" content="width=device-width, initial-scan=1.0">
					<meta name="description" content="Example of HTML5 metadata">
					<meta name="keywords" content=HTML5, metadata, tutorial">
					<meta name="author" content="Saide Hossain">
					<!-- Base URL -->
					<base href="https://example.com/">
					<!-- External resources -->
					<link rel="icon" href="favicon.ico">
					<link rel="stylesheet" href="styles.css">
					<!-- Internal CSS -->
					<style>
						body {
							font-family: Arial, sans-serif;
						}
					</style>
					<script>
						console.log("Page loaded");
					</script>
					</head>
					<body>
						<h1>Metadata Example Page</h1>
					</body>
					</html>
