HTML5 ফান্ডামেন্টালস

HTML হলো ওয়েবপেজ তৈরী করার স্ট্যান্ডার্ড ল্যাংগুয়েজ। HTML5 is the latest version, যা modern features এবং multimedia এর জন্য better support দেয়। প্রতিটি HTML document একটি basic structure follow করে: doctype, html, head, এবং body। Web pages তৈরি হয় elements (tags) ব্যবহার করে, যা content define করে। Attributes elements-এর অতিরিক্ত information প্রদান করে। Elements একটির ভিতরে আরেকটি nested হয়ে structure তৈরি করতে পারে। Comments কোডের ভিতরে note যোগ করতে ব্যবহার করা হয়, যা browser-এ প্রদর্শিত হয় না।

Example:

				
					<!DOCTYPE html>
					<html lang="en">
					<head>
					<meta charset="UTF-8">
					<title>My First Page</title>
					</head>
					<body>
					<!-- This is a heading -->
					<h1 class="title">Hello World</h1>
					<!-- This is a paragraph -->
					<p>This is my first HTML5 page.</p>
					</body>
					</html>

ডকুমেন্ট মেটাডাটা

head element ডকুমেন্টের মেটাডাটা ধারণ করে, যা পেজ সম্পর্কে অতিরিক্ত তথ্য ব্রাউজারকে জানায়। title ব্রাউজার ট্যাব-এ প্রদর্শিত পেজের নাম নির্ধারণ করে। meta tags ক্যারেক্টার এনকোডিং, রেসপনসিভনেস এবং এসইও সম্পর্কিত গুরুত্বপূর্ণ তথ্য প্রদান করে। base tag সব রিলেটিভ লিংকের জন্য একটি বেস ইউআরএল নির্ধারণ করে। link tag এক্সটার্নাল রিসোর্স যেমন স্টাইলশিট এবং ফ্যাভিকন যুক্ত করতে ব্যবহার করা হয়। style tag ইন্টারনাল সিএসএস লেখার জন্য ব্যবহার করা হয়। script tag পেজে জাভাস্ক্রিপ্ট যোগ করার জন্য ব্যবহার করা হয়।

Example:

					
					<!DOCTYPE html>
					<html lang="en">
					<head>
					<!-- Page title -->
					<title>Document Metadata Example</title>
					<!-- Meta tags -->
					<meta charset="UTF-8">
					<meta name="viewport" content="width=device-width, initial-scan=1.0">
					<meta name="description" content="Example of HTML5 metadata">
					<meta name="keywords" content=HTML5, metadata, tutorial">
					<meta name="author" content="Saide Hossain">
					<!-- Base URL -->
					<base href="https://example.com/">
					<!-- External resources -->
					<link rel="icon" href="favicon.ico">
					<link rel="stylesheet" href="styles.css">
					<!-- Internal CSS -->
					<style>
						body {
							font-family: Arial, sans-serif;
						}
					</style>
					<script>
						console.log("Page loaded");
					</script>
					</head>
					<body>
						<h1>Metadata Example Page</h1>
					</body>
					</html>

টেক্সড কন্টেন্ট ইলিমেন্টস

টেক্সট কন্টেন্ট ইলিমেন্টস ওয়েবপেজে লেখা (text) সুন্দরভাবে দেখানো এবং সাজানোর জন্য ব্যবহার করা হয়। হেডিংস (h1-h6) বিভিন্ন স্তরের শিরোনাম তৈরি করতে ব্যবহার করা হয়। প্যারাগ্রাফ (p) সাধারণ লেখা প্রদর্শনের জন্য ব্যবহৃত হয়। লাইন ব্রেক (br) একই লাইনের মধ্যে নতুন লাইন তৈরি করে। হরাইজোন্টাল রুল (hr) একটি দাগ (line) তৈরি করে, যা কন্টেন্ট আলাদা করতে সাহায্য করে। প্রিফরমেটেড টেক্সট (pre) লেখার ফরম্যাট (space, line break) ঠিক রাখে। ব্লককুয়োট (blockquote) বড় উক্তি বা quotation দেখানোর জন্য ব্যবহৃত হয়। ইনলাইন কোটেশন (q) ছোট উক্তি দেখানোর জন্য ব্যবহৃত হয়। অ্যাব্রিভিয়েশন (abbr) সংক্ষিপ্ত শব্দের পূর্ণরূপ বোঝাতে ব্যবহার করা হয়। অ্যাড্রেস (address) যোগাযোগ সংক্রান্ত তথ্য দেখাতে ব্যবহার করা হয়। সাইট (cite) কোনো বই, আর্টিকেল বা কাজের নাম বোঝাতে ব্যবহার করা হয়। কোড (code) প্রোগ্রামিং কোড দেখানোর জন্য ব্যবহৃত হয়। মার্ক (mark) কোনো লেখা হাইলাইট করতে ব্যবহার করা হয়। স্মল (small) ছোট লেখা দেখানোর জন্য ব্যবহৃত হয়। স্ট্রং (strong) গুরুত্বপূর্ণ লেখা বোঝাতে ব্যবহার করা হয়। ইম (em) কোনো লেখায় জোর বা গুরুত্ব দিতে ব্যবহার করা হয়। সাবস্ক্রিপ্ট (sub) নিচের দিকে ছোট লেখা দেখায় (যেমন: H₂O)। সুপারস্ক্রিপ্ট (sup) উপরের দিকে ছোট লেখা দেখায় (যেমন: x²)। টাইম (time) তারিখ বা সময় বোঝাতে ব্যবহার করা হয়। ডেট (date) তারিখ সম্পর্কিত তথ্য দেখাতে ব্যবহৃত হয়।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <title>Text Content Example</title> </head> <body> <h1>এটি একটি হেডিং</h1> <p>এটি একটি প্যারাগ্রাফ।<br>এখানে লাইন ব্রেক হয়েছে।</p> <hr> <pre> এই লেখা যেমন লিখা হয়েছে ঠিক তেমনই দেখাবে। </pre> <blockquote> এটি একটি বড় কোটেশন। </blockquote> <p>সে বলল <q>এটি ছোট কোটেশন</q></p> <p><abbr title="HyperText Markup Language">HTML</abbr> একটি ওয়েব ল্যাংগুয়েজ</p> <address> Email: example@mail.com </address> <p><cite>Book Name</cite> এখানে ব্যবহার করা হয়েছে</p> <p>কোড: <code>console.log("Hello")</code></p> <p>এটি <mark>হাইলাইট</mark> করা টেক্সট</p> <p><small>ছোট লেখা</small></p> <p><strong>গুরুত্বপূর্ণ</strong> এবং <em>জোর দেওয়া</em> লেখা</p> <p>H<sub>2</sub>O এবং x<sup>2</sup></p> <p>সময়: <time datetime="2026-03-18">১৮ মার্চ ২০২৬</time></p> </body> </html>

গ্রুপিং কন্টেন্ট

এই ইলিমেন্টগুলো ওয়েবপেজের স্ট্রাকচার এবং কন্টেন্ট গ্রুপিং করার জন্য ব্যবহার করা হয়। ডিভ (div) একটি সাধারণ কন্টেইনার, যা বিভিন্ন কন্টেন্ট একসাথে গ্রুপ করতে ব্যবহার করা হয়। স্পেন (span) ইনলাইন কন্টেন্ট গ্রুপ করতে ব্যবহার করা হয়। সেকশন (section) পেজের একটি নির্দিষ্ট অংশ বা সেকশন বোঝাতে ব্যবহার করা হয়। আর্টিকেল (article) স্বতন্ত্র বা আলাদা কন্টেন্ট (যেমন: ব্লগ পোস্ট, নিউজ) বোঝাতে ব্যবহৃত হয়। নেভ (nav) নেভিগেশন লিংকগুলো রাখার জন্য ব্যবহার করা হয়। এসাইড (aside) মূল কন্টেন্টের পাশের অতিরিক্ত তথ্য (sidebar) দেখাতে ব্যবহৃত হয়। হেডার (header) পেজ বা সেকশনের উপরের অংশ বোঝায় (যেমন: title, logo)। ফুটার (footer) পেজ বা সেকশনের নিচের অংশ বোঝায় (যেমন: copyright, contact info)। মেইন (main) পেজের মূল (main) কন্টেন্ট ধারণ করে।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <title>Grouping Elements</title> </head> <body> <header> <h1>আমার ওয়েবসাইট</h1> </header> <nav> <a href="#">Home</a> <a href="#">About</a> </nav> <main> <section> <h2>সেকশন ১</h2> <p>এটি একটি সেকশন</p> </section> <article> <h2>আর্টিকেল</h2> <p>এটি একটি আর্টিকেল কন্টেন্ট</p> </article> <aside> <p>এটি সাইডবার কন্টেন্ট</p> </aside> <div> <p>এটি div দিয়ে গ্রুপ করা হয়েছে</p> </div> <p>এটি একটি <span style="color:red;">inline span</span> উদাহরণ</p> </main> <footer> <p>© 2026 আমার ওয়েবসাইট</p> </footer> </body> </html>

লিস্ট

লিস্ট ইলিমেন্টগুলো ওয়েবপেজে তথ্যকে তালিকার আকারে দেখানোর জন্য ব্যবহার করা হয়। অর্ডার লিস্ট (ol) ব্যবহার করা হয় যেখানে আইটেমগুলোকে সংখ্যার ক্রমে দেখাতে হয়। আনওর্ডার লিস্ট (ul) ব্যবহার করা হয় যেখানে আইটেমগুলো বুলেট বা ডট আকারে প্রদর্শিত হয়। লিস্ট আইটেম (li) প্রতিটি তালিকার একটি আইটেমকে নির্দেশ করে। ডিস্ক্রিপশন লিস্ট (dl) শর্ত বা টার্ম এবং তার ব্যাখ্যা প্রদর্শনের জন্য ব্যবহৃত হয়, যেখানে dt শর্ত/টার্ম এবং dd তার বর্ণনা বা ব্যাখ্যা বোঝায়।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>লিস্ট উদাহরণ</title> </head> <body> <h1>লিস্ট ইলিমেন্ট উদাহরণ</h1> <h2>অর্ডার লিস্ট (Ordered List)</h2> <ol> <li>আপেল</li> <li>কলা</li> <li>আম</li> </ol> <h2>আনওর্ডার লিস্ট (Unordered List)</h2> <ul> <li>লেবু</li> <li>কমলা</li> <li>আনারস</li> </ul> <h2>ডিস্ক্রিপশন লিস্ট (Description List)</h2> <dl> <dt>HTML</dt> <dd>ওয়েবপেজ তৈরি করার মার্কআপ ল্যাঙ্গুয়েজ।</dd> <dt>CSS</dt> <dd>ওয়েবপেজের স্টাইল এবং লেআউটের জন্য ব্যবহৃত হয়।</dd> <dt>JavaScript</dt> <dd>ওয়েবপেজকে ইন্টারঅ্যাকটিভ করার জন্য ব্যবহৃত হয়।</dd> </dl> </body> </html>

লিংক এবং নেভিগেশন ওয়েবপেজে ব্যবহারকারীদের জন্য পেজ এবং কন্টেন্টের মধ্যে ন্যাভিগেশন সহজ করে। অ্যাঙ্কর ট্যাগ (a) হলো লিংক তৈরির প্রধান ইলিমেন্ট। ইন্টারনাল লিংক পেজের ভেতরের অন্য কোনো অংশে নেভিগেট করে, আর এক্সটারনাল লিংক অন্য ওয়েবসাইটে নিয়ে যায়। ইমেইল লিংক (mailto) সরাসরি ইমেইল প্রোগ্রামে খোলে, এবং টেলিফোন লিংক (tel) ফোন কল করার জন্য ব্যবহার হয়। টার্গেট অ্যাট্রিবিউট লিংক কোন উইন্ডো বা ট্যাবে খোলা হবে তা নির্ধারণ করে। ডাউনলোড অ্যাট্রিবিউট ব্যবহার করে লিংক করা ফাইল ডাউনলোড করা যায়।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>লিংক এবং নেভিগেশন উদাহরণ</title> </head> <body> <h1>লিংক এবং নেভিগেশন উদাহরণ</h1> <h2>ইন্টারনাল লিংক</h2> <p><a href="#section2">সেকশন ২-এ যান</a></p> <h2>এক্সটারনাল লিংক</h2> <p><a href="https://www.google.com" target="_blank">Google-এ যান</a></p> <h2>ইমেইল লিংক</h2> <p><a href="mailto:example@example.com">আমাকে ইমেইল পাঠান</a></p> <h2>টেলিফোন লিংক</h2> <p><a href="tel:+880123456789">আমাকে কল করুন</a></p> <h2>ডাউনলোড লিংক</h2> <p><a href="sample.pdf" download>Sample PDF ডাউনলোড করুন</a></p> <h2 id="section2">সেকশন ২</h2> <p>এটি সেকশন ২। আপনি উপরের ইন্টারনাল লিংক থেকে এখানে এসেছেন।</p> </body> </html>

ইমেইজ এন্ড মিডিয়া

ইমেজ এবং মিডিয়া ইলিমেন্ট ওয়েবপেজকে ভিজ্যুয়ালি সমৃদ্ধ করে। img ট্যাগ দিয়ে ইমেজ দেখানো হয় এবং এর জন্য src, alt, width, height মতো অ্যাট্রিবিউট ব্যবহার করা হয়। রিসপনসিভ ইমেজের জন্য srcset ও sizes ব্যবহার করা হয়। figure ও figcaption ইমেজের সাথে ক্যাপশন দেখাতে ব্যবহৃত হয়। picture ইলিমেন্ট দিয়ে বিভিন্ন স্ক্রিন সাইজের জন্য ভিন্ন ইমেজ লোড করা যায়। অডিও (audio) এবং ভিডিও (video) ইলিমেন্ট ওয়েবপেজে মিডিয়া প্লে করতে সাহায্য করে, যেখানে source দিয়ে ফাইল সংযুক্ত করা হয় এবং track সাবটাইটেল বা ক্যাপশন দেখাতে ব্যবহৃত হয়। আইফ্রেম (iframe) দিয়ে অন্য ওয়েবপেজ বা মিডিয়া এম্বেড করা যায়।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>ইমেজ এবং মিডিয়া উদাহরণ</title> </head> <body> <h1>ইমেজ এবং মিডিয়া উদাহরণ</h1> <h2>ইমেজ ট্যাগ এবং অ্যাট্রিবিউট</h2> <img src="example.jpg" alt="উদাহরণ ইমেজ" width="300" height="200"> <h2>রিসপনসিভ ইমেজ</h2> <picture> <source srcset="example-large.jpg" media="(min-width: 800px)"> <source srcset="example-small.jpg" media="(max-width: 799px)"> <img src="example.jpg" alt="রিসপনসিভ ইমেজ"> </picture> <h2>ফিগার এবং ফিগক্যাপশন</h2> <figure> <img src="example.jpg" alt="ফিগার উদাহরণ" width="300"> <figcaption>এটি ফিগার এবং ফিগক্যাপশন উদাহরণ।</figcaption> </figure> <h2>অডিও উদাহরণ</h2> <audio controls> <source src="example.mp3" type="audio/mpeg"> আপনার ব্রাউজার অডিও সাপোর্ট করে না। </audio> <h2>ভিডিও উদাহরণ</h2> <video width="320" height="240" controls> <source src="example.mp4" type="video/mp4"> <track src="example.vtt" kind="subtitles" srclang="bn" label="Bangla"> আপনার ব্রাউজার ভিডিও সাপোর্ট করে না। </video> <h2>আইফ্রেম উদাহরণ</h2> <iframe src="https://www.example.com" width="400" height="300" title="Example Iframe"></iframe> </body> </html>

টেবল

টেবল ইলিমেন্ট ওয়েবপেজে ডাটা সুন্দরভাবে গ্রিড আকারে প্রদর্শনের জন্য ব্যবহৃত হয়। table হলো মূল টেবল কন্টেইনার। প্রতিটি রো তৈরি করতে tr ব্যবহার হয়, কলাম বা সেল দেখানোর জন্য td (ডাটা সেল) এবং th (হেডার সেল) ব্যবহার হয়। টেবলকে আরও সঠিকভাবে গঠন করতে thead, tbody, এবং tfoot ব্যবহার করা হয়। টেবলের জন্য শিরোনাম বা বিবরণ দিতে caption ব্যবহার করা হয়। সেল মেশানোর জন্য rowspan (উল্লম্ব) এবং colspan (অনুভূমিক) ব্যবহার করা হয়। টেবলটির ব্যাকগ্রাউন্ড, লেবেল বা স্কোপ সংজ্ঞায় scope অ্যাট্রিবিউট ব্যবহার করা হয়।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>টেবল উদাহরণ</title> </head> <body> <h1>টেবল উদাহরণ</h1> <table> <caption>ছাত্রদের মার্কশিট</caption> <thead> <tr> <th scope="col">নাম</th> <th scope="col">বিষয়</th> <th scope="col">মার্কস</th> </tr> </thead> <tbody> <tr> <td rowspan="2">রহিম</td> <td>গণিত</td> <td>85</td> </tr> <tr> <td>বিজ্ঞান</td> <td>90</td> </tr> <tr> <td>করিম</td> <td colspan="2">প্রকৌশল বিষয় (অংশগ্রহণ)</td> </tr> </tbody> <tfoot> <tr> <td colspan="3">মোট শিক্ষার্থী: 3</td> </tr> </tfoot> </table> </body> </html>

সিম্যান্টিক HTML5

সিম্যান্টিক HTML5 হলো এমন ইলিমেন্টস যা কন্টেন্টের অর্থ এবং কাঠামো স্পষ্টভাবে বোঝাতে সাহায্য করে। এর মাধ্যমে ব্রাউজার, সার্চ ইঞ্জিন এবং অ্যাক্সেসিবিলিটি টুলস কন্টেন্ট বুঝতে পারে। উদাহরণস্বরূপ, সেকশন ও আর্টিকেল পেজের বিভিন্ন অংশ আলাদা করে দেখায়, যেখানে আর্টিকেল স্বতন্ত্র কন্টেন্ট বোঝায়। হেডার এবং ফুটার পেজ বা সেকশনের উপরের ও নিচের অংশ বোঝাতে ব্যবহার হয়। নেভ নেভিগেশন লিঙ্কের জন্য, এসাইড পাশে অতিরিক্ত কন্টেন্টের জন্য এবং মেইন পেজের প্রধান কন্টেন্টের জন্য ব্যবহৃত হয়। ফিগার এবং ফিগক্যাপশন ছবি বা মিডিয়া এবং তার বর্ণনা প্রদর্শনের জন্য ব্যবহৃত হয়।

মালটিমিডিয়া এবং এমবেডেড

মালটিমিডিয়া এবং এমবেডেড ইলিমেন্টগুলো ওয়েবপেজে অডিও, ভিডিও, এবং অন্যান্য এক্সটার্নাল কন্টেন্ট প্রদর্শনের জন্য ব্যবহার করা হয়। এতে ব্যবহারকারীরা সরাসরি ব্রাউজারে মিডিয়া কন্টেন্ট শুনতে বা দেখতে পারে। প্রধান ইলিমেন্টগুলো হলো audio, video, iframe, embed, এবং object। অডিও/ভিডিওতে controls, autoplay, loop, muted অ্যাট্রিবিউট ব্যবহার করা হয়। iframe দিয়ে YouTube ভিডিও, Google Maps বা অন্য ওয়েব কন্টেন্ট এমবেড করা যায়।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <title>মালটিমিডিয়া এবং এমবেডেড উদাহরণ</title> </head> <body> <h1>মালটিমিডিয়া উদাহরণ</h1> <h2>অডিও প্লেয়ার</h2> <audio controls autoplay loop> <source src="music.mp3" type="audio/mpeg"> আপনার ব্রাউজার অডিও ট্যাগ সাপোর্ট করে না। </audio> <h2>ভিডিও প্লেয়ার</h2> <video width="320" height="240" controls muted> <source src="video.mp4" type="video/mp4"> আপনার ব্রাউজার ভিডিও ট্যাগ সাপোর্ট করে না। </video> <h2>YouTube ভিডিও এমবেড</h2> <iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" title="YouTube video player" frameborder="0" allowfullscreen></iframe> <h2>এমবেড এবং অবজেক্ট উদাহরণ</h2> <embed src="example.pdf" type="application/pdf" width="500" height="400"> <object data="example.pdf" type="application/pdf" width="500" height="400"> আপনার ব্রাউজার PDF দেখাতে সমর্থ নয়। </object> </body> </html>

গ্রাফিক্স

ওয়েবপেজে গ্রাফিক্স তৈরি করতে HTML5 এ Canvas এবং SVG (Scalable Vector Graphics) ব্যবহার করা হয়।

  • Canvas একটি ক্যানভাস এলাকা প্রদান করে যেখানে JavaScript দিয়ে ড্রয়িং করা যায়। এটি পিক্সেল-বেসড গ্রাফিক্স।
  • বেসিক ক্যানভাস ড্রয়িং এ লাইন্স, আর্ক, রেক্টেঙ্গল, সার্কেল ইত্যাদি আঁকা হয়।
  • SVG হলো ভেক্টর বেসড গ্রাফিক্স যা স্কেল করা যায় এবং HTML এর মধ্যে সরাসরি ইনলাইন করা যায়।
  • ইনলাইন SVG দিয়ে HTML এর মধ্যে সরাসরি shapes, paths, এবং texts যোগ করা যায়।

Example:

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <title>গ্রাফিক্স উদাহরণ</title> </head> <body> <h1>Canvas উদাহরণ</h1> <canvas id="myCanvas" width="300" height="150" style="border:1px solid #000000;"> আপনার ব্রাউজার ক্যানভাস সাপোর্ট করে না। </canvas> <script> var canvas = document.getElementById("myCanvas"); var ctx = canvas.getContext("2d"); ctx.fillStyle = "#FF0000"; ctx.fillRect(20, 20, 100, 50); // লাল রেক্টেঙ্গল ctx.beginPath(); ctx.arc(150, 75, 40, 0, 2 * Math.PI); // সার্কেল ctx.stroke(); </script> <h1>SVG উদাহরণ</h1> <svg width="300" height="150" style="border:1px solid #000000;"> <rect x="10" y="10" width="100" height="50" fill="blue" /> <circle cx="200" cy="75" r="40" stroke="green" stroke-width="4" fill="yellow" /> <line x1="0" y1="0" x2="300" y2="150" stroke="red" stroke-width="2" /> </svg> </body> </html>

APIs in HTML5

HTML5 বিভিন্ন নতুন APIs প্রদান করে যা ওয়েব অ্যাপ্লিকেশনকে আরও ইন্টারেকটিভ ও শক্তিশালী করে:

  • জিওলোকেশন API → ব্যবহারকারীর লোকেশন (latitude, longitude) জানার জন্য।
  • ড্রাগ এবং ড্রপ API → ড্র্যাগেবল এলিমেন্ট তৈরি ও ড্রপ করা।
  • ওয়েব স্টোরেজ API → localStorage ও sessionStorage দিয়ে ব্রাউজারে ডাটা স্টোর করা।
  • ওয়েব ওয়ার্কার্স → ব্যাকগ্রাউন্ড থ্রেডে জাভাস্ক্রিপ্ট রান করানো।
  • Server-Sent Events (SSE) → সার্ভার থেকে ক্লায়েন্টে রিয়েল-টাইম আপডেট।
  • WebSockets → দুই-পক্ষের রিয়েল-টাইম কমিউনিকেশন।

HTML5 ফর্ম ভেলিডেশন ব্যবহার করে ইউজার ইনপুট সঠিক কিনা চেক করা যায়। এর মধ্যে তিনটি প্রধান ধাপ রয়েছে:

  • বিল্ট-ইন ভেলিডেশন → required, type="email", min, max ইত্যাদি ব্যবহার করে স্বয়ংক্রিয় চেক।
  • কন্সট্রেইন্ট ভেলিডেশন API → জাভাস্ক্রিপ্ট দিয়ে ফিল্ডের validity এবং checkValidity() চেক করা।
  • কাস্টম ভেলিডেশন → নিজস্ব নিয়ম দিয়ে ইনপুট যাচাই ও কাস্টম মেসেজ দেখানো।

<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <title>ফর্ম ভেলিডেশন উদাহরণ</title> </head> <body> <h1>বিল্ট-ইন ভেলিডেশন</h1> <form> <label>ইমেইল: <input type="email" required placeholder="email@example.com"></label> <br><br> <label>পাসওয়ার্ড: <input type="password" required minlength="6"></label> <br><br> <button type="submit">সাবমিট</button> </form> <hr> <h1>কাস্টম ভেলিডেশন</h1> <form id="customForm"> <label>ইউজারনেম (5+ অক্ষর): <input type="text" id="username" required></label> <br><br> <button type="submit">সাবমিট</button> </form> <p id="errorMsg" style="color:red;"></p> <script> document.getElementById("customForm").addEventListener("submit", function(e){ var username = document.getElementById("username").value; if(username.length < 5){ e.preventDefault(); document.getElementById("errorMsg").innerText = "ইউজারনেম অন্তত 5 অক্ষরের হতে হবে।"; } else { document.getElementById("errorMsg").innerText = ""; } }); </script> </body> </html>
